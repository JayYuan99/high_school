# 3.3 牛顿第三定律

## 本节学习目标

学完本节，需要能做到：

- 理解力的作用是相互的。
- 知道什么是作用力和反作用力。
- 掌握牛顿第三定律的内容。
- 会判断一对力是否为作用力和反作用力。
- 会区分作用力与反作用力和平衡力。
- 能从相互作用角度进行初步受力分析。
- 会按重力、弹力、摩擦力的顺序分析简单物体受力。

## 核心知识点讲解

### 1. 力总是成对出现

力是物体对物体的作用。只要一个物体对另一个物体施加力，后一个物体也一定同时对前一个物体施加力。

例如：

- 手拉弹簧，手给弹簧拉力，弹簧也给手拉力。
- 人推桌子，桌子也推人。
- 地球吸引物体，物体也吸引地球。
- 脚向后蹬地，地面向前推脚。

这说明力的作用是相互的。

### 2. 作用力和反作用力

两个物体之间相互作用的一对力，叫作作用力和反作用力。

其中哪一个叫作用力、哪一个叫反作用力，并没有绝对规定。把其中一个叫作用力，另一个就叫反作用力。

例如，人推墙：

- 人对墙的推力。
- 墙对人的推力。

这两个力就是一对作用力和反作用力。

### 3. 牛顿第三定律

牛顿第三定律内容：

```text
两个物体之间的作用力和反作用力总是大小相等、方向相反，作用在同一条直线上。
```

可以概括为：

```text
等大、反向、共线、异物、同性质、同时存在。
```

其中：

- 等大：大小相等。
- 反向：方向相反。
- 共线：作用在同一直线上。
- 异物：分别作用在两个不同物体上。
- 同性质：例如都是弹力、都是摩擦力、都是引力。
- 同时存在：一个消失，另一个也同时消失。

### 4. 大人和小孩掰手腕时谁的力大

大人容易把小孩手压下去，不是因为大人对小孩的力大于小孩对大人的力。

根据牛顿第三定律：

```text
大人对小孩的力 = 小孩对大人的力
```

它们大小相等、方向相反。

小孩手被压下去，是因为小孩手自身还受到其他力的作用，且小孩肌肉等不能提供足够支撑来保持平衡。

不要把“谁赢了”和“相互作用力是否相等”混在一起。

### 5. 作用力和反作用力不能相互抵消

作用力和反作用力分别作用在两个不同物体上，因此不能在同一个物体的受力分析中相互抵消。

例如，书压桌面，桌面支持书：

- 书对桌面的压力作用在桌子上。
- 桌面对书的支持力作用在书上。

它们作用在不同物体上，不能说互相平衡。

### 6. 平衡力

如果一个物体同时受到两个力作用，这两个力大小相等、方向相反、作用在同一直线上，并且作用在同一物体上，那么这两个力是一对平衡力。

例如，书静止在桌面上：

- 书受到重力。
- 书受到桌面的支持力。

这两个力都作用在书上，大小相等、方向相反，是一对平衡力。

### 7. 作用力反作用力和平衡力的区别

| 比较项目 | 作用力和反作用力 | 平衡力 |
| --- | --- | --- |
| 作用对象 | 两个不同物体 | 同一个物体 |
| 是否同性质 | 一定同性质 | 不一定同性质 |
| 是否同时产生消失 | 一定同时 | 不一定 |
| 能否相互抵消 | 不能 | 可以使合力为 0 |
| 例子 | 书压桌面和桌面受压力 | 书受重力和支持力 |

判断时最重要的一句话：

```text
同物平衡，异物相互。
```

### 8. 初步受力分析方法

分析物体受力时，必须先明确研究对象。

常用顺序：

1. 重力：地球对物体的吸引，方向竖直向下。
2. 弹力：看物体与谁接触、哪里被挤压或拉伸。
3. 摩擦力：看接触面是否粗糙，是否有相对运动或趋势。

只画研究对象受到的力，不画研究对象对别的物体施加的力。

例如木块静止在粗糙斜面上：

- 木块受重力 `G`，竖直向下。
- 木块受斜面支持力 `FN`，垂直斜面向上。
- 木块有沿斜面下滑趋势，受静摩擦力 `Ff`，沿斜面向上。

## 重点梳理

### 牛顿第三定律

```text
作用力和反作用力总是大小相等、方向相反，作用在同一条直线上。
```

### 六个关键词

```text
等大、反向、共线、异物、同性质、同时
```

### 判断口诀

```text
同物平衡，异物相互。
```

### 受力分析顺序

```text
先重力，再弹力，后摩擦力。
```

## 难点突破

### 难点 1：作用力和反作用力大小相等，为什么物体还会动

因为这两个力作用在不同物体上。

物体是否运动改变，要看它自己所受各力的合力，而不是看它对别的物体施加了什么力。

例如人推箱子，箱子是否运动取决于箱子受到的人推力、摩擦力等的合力。

### 难点 2：马拉车，车也拉马，为什么车会前进

马拉车的力和车拉马的力是一对作用力和反作用力，大小相等，作用在不同物体上。

马和车组成的系统能前进，关键还要看地面对马的静摩擦力。马向后蹬地，地面对马产生向前的静摩擦力，推动马和车前进。

### 难点 3：重力和支持力是不是作用力反作用力

不是。

书静止在桌面上时：

- 书受重力，施力物体是地球。
- 书受支持力，施力物体是桌面。

这两个力都作用在书上，是平衡力，不是作用力和反作用力。

书所受支持力的反作用力是书对桌面的压力。

### 难点 4：一对作用力和反作用力一定是同种力吗

是。

例如桌面对书的支持力和书对桌面的压力都是弹力；地球对人的重力和人对地球的引力都是引力。

平衡力不一定同种，例如静止书本受到的重力和支持力就是不同性质的力。

### 难点 5：受力分析为什么不能画“物体对外的力”

受力分析只分析研究对象受到的力。

如果研究对象是木块，就只画别的物体对木块的力，不画木块对桌面、地球、绳子的力。

画错对象，是受力分析最常见错误。

## 例题讲解

### 例题 1：判断作用力和反作用力

题目：

书放在水平桌面上。指出“桌面对书的支持力”的反作用力。

分析：

作用力和反作用力发生在同两个物体之间，性质相同，方向相反。

答案：

桌面对书的支持力的反作用力是书对桌面的压力。

反思：

不是书受到的重力。重力的施力物体是地球，支持力的施力物体是桌面。

### 例题 2：区分平衡力和相互作用力

题目：

吊灯静止悬挂。吊灯受到重力和绳子的拉力，这两个力是什么关系？绳子对吊灯的拉力的反作用力是什么？

答案：

吊灯受到的重力和绳子的拉力作用在同一物体吊灯上，是一对平衡力。

绳子对吊灯拉力的反作用力是吊灯对绳子的拉力。

### 例题 3：斜面上木块受力分析

题目：

木块静止在粗糙斜面上，画出木块受力并说明方向。

分析：

研究对象是木块。按重力、弹力、摩擦力顺序分析。

答案：

木块受到三个力：

- 重力 `G`：竖直向下。
- 支持力 `FN`：垂直斜面向外。
- 静摩擦力 `Ff`：沿斜面向上，因为木块有沿斜面下滑趋势。

### 例题 4：人走路中的相互作用

题目：

人向前走路时，脚受到地面的摩擦力方向如何？这对相互作用力是哪一对？

分析：

脚向后蹬地，脚相对地面有向后滑的趋势，所以地面对脚的静摩擦力向前。

答案：

地面对脚的静摩擦力向前。它的反作用力是脚对地面的静摩擦力，方向向后。

## 易错点整理

### 易错点 1：认为作用力和反作用力能抵消

它们作用在不同物体上，不能抵消。

### 易错点 2：把重力和支持力当成作用力反作用力

它们作用在同一物体上，是平衡力。

### 易错点 3：只看大小方向，不看作用对象

判断力的关系必须看作用对象。

### 易错点 4：受力分析时画上反作用力

研究对象只画受到的力，不画它施加给别人的力。

### 易错点 5：认为力有先后

作用力和反作用力同时产生、同时消失，没有先后。

## 考点考证点整理

### 考点 1：牛顿第三定律表述

常考填空和判断，关键词是等大、反向、共线、异物。

### 考点 2：寻找反作用力

先找原力的施力物体和受力物体，再把二者互换。

### 考点 3：平衡力与相互作用力辨析

核心看是否作用在同一个物体上。

### 考点 4：简单受力分析

常考斜面木块、叠放物体、悬挂物、桌面物体。

### 考点 5：生活应用

走路、划船、汽车驱动、火箭喷气都体现相互作用。

11. 有人说：“作用力和反作用力大小相等、方向相反，所以一定平衡。”这个说法为什么错？

13. 粗糙水平桌面上，B 木块受水平拉力但仍静止，A 木块叠放在 B 上且也静止。分析 B 木块受到的主要力。

## 练习题答案

### 基础题答案

1. 两个物体之间的作用力和反作用力总是大小相等、方向相反，作用在同一直线上。

2. 不，它们分别作用在两个不同物体上。

3. 书对桌面的压力。

4. 是一对平衡力。

5. 平衡力作用在同一物体上；作用力和反作用力作用在两个不同物体上。

### 巩固题答案

6. 一样大，方向相反。

7. 苹果也吸引地球，二者是作用力和反作用力，大小相等、方向相反。

8. 猴子受重力和树枝对它的拉力。反作用力是猴子对树枝的拉力。

9. 通常受三个力：重力、斜面支持力、沿斜面向上的静摩擦力。

10. 对汽车驱动轮，地面的静摩擦力方向向前。

### 提升题答案

11. 错在忽略作用对象。作用力和反作用力分别作用在两个不同物体上，不能在同一物体上合成，也不能相互平衡。

12. 油桶受重力和汽车对它的支持力。重力的反作用力是油桶对地球的引力；支持力的反作用力是油桶对汽车的压力。

13. B 受到重力、桌面对 B 的支持力、A 对 B 的压力、水平拉力、桌面对 B 的静摩擦力；若 A 与 B 之间没有相对运动趋势，A 对 B 可不产生水平摩擦力，具体要看题目条件。
