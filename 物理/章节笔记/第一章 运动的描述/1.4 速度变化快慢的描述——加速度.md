# 1.4 速度变化快慢的描述——加速度

## 本节学习目标

学完本节，需要能做到：

- 理解加速度是描述速度变化快慢的物理量。
- 会用公式 `a = Δv / Δt` 计算加速度。
- 知道加速度的单位是 `m/s^2`，并能说出单位的含义。
- 理解加速度是矢量，方向与速度变化量 `Δv` 的方向相同。
- 能区分速度、速度变化量和加速度。
- 会判断直线运动中加速、减速时加速度与速度方向的关系。
- 能从 `v-t` 图像的斜率判断加速度大小和方向。
- 理解“变化率”的思想：加速度是速度对时间的变化率。

## 核心知识点讲解

### 1. 为什么要引入加速度

速度可以描述物体运动得快不快，但速度不能直接描述“速度变得快不快”。

例如：

- 小汽车从静止加速到 `100 km/h`，用时 `10 s`。
- 火车从静止加速到 `100 km/h`，用时 `300 s`。

两者的速度变化量相同，都是从 `0` 增大到 `100 km/h`，但小汽车明显“提速更快”。

这说明只看速度大小或速度变化量还不够，还要看速度变化用了多长时间。

物理学中用**加速度**描述速度变化的快慢。

### 2. 加速度的定义

加速度表示速度变化的快慢。

如果物体在时间 `Δt` 内速度变化量为 `Δv`，则加速度为：

```text
a = Δv / Δt
```

其中：

- `a` 表示加速度。
- `Δv` 表示速度变化量。
- `Δt` 表示发生这段速度变化所用的时间。

速度变化量为：

```text
Δv = v2 - v1
```

其中：

- `v1` 表示初速度。
- `v2` 表示末速度。

所以也可以写成：

```text
a = (v2 - v1) / Δt
```

这个公式的直观意思是：

```text
单位时间内速度变化越多，加速度越大。
```

例如，汽车在 `5 s` 内速度从 `2 m/s` 增大到 `17 m/s`：

```text
Δv = 17 m/s - 2 m/s = 15 m/s
a = Δv / Δt = 15 m/s / 5 s = 3 m/s^2
```

表示汽车的速度平均每秒增加 `3 m/s`。

### 3. 加速度的单位

加速度的国际单位是米每二次方秒，符号为：

```text
m/s^2
```

也可以写成：

```text
m·s^-2
```

为什么是这个单位？

因为：

```text
a = Δv / Δt
```

而速度单位是 `m/s`，时间单位是 `s`，所以：

```text
(m/s) / s = m/s^2
```

`1 m/s^2` 的含义是：

```text
速度每秒变化 1 m/s
```

例如，加速度 `2 m/s^2` 可以理解为：如果加速度保持不变，物体速度每秒增加或减少 `2 m/s`。

### 4. 加速度是矢量

加速度不仅有大小，还有方向，所以加速度是矢量。

加速度的方向与速度变化量 `Δv` 的方向相同，而不是一定与速度方向相同。

这一点非常关键。

在直线运动中，先规定正方向，然后用正负号表示方向：

```text
Δv = v2 - v1
a = Δv / Δt
```

因为 `Δt` 总是正的，所以：

- `Δv > 0`，则 `a > 0`，加速度沿正方向。
- `Δv < 0`，则 `a < 0`，加速度沿负方向。

也就是说，判断加速度方向的核心是看速度变化量 `Δv` 的方向。

### 5. 加速运动和减速运动中的方向关系

在直线运动中，可以用速度方向和加速度方向的关系判断物体是加速还是减速。

如果速度和加速度方向相同：

```text
速度大小增大，物体做加速运动。
```

如果速度和加速度方向相反：

```text
速度大小减小，物体做减速运动。
```

例如，规定向东为正方向：

- 汽车向东运动，`v > 0`，如果 `a > 0`，速度越来越大。
- 汽车向东运动，`v > 0`，如果 `a < 0`，速度越来越小。
- 汽车向西运动，`v < 0`，如果 `a < 0`，速度大小越来越大。
- 汽车向西运动，`v < 0`，如果 `a > 0`，速度大小越来越小。

所以不能简单地说“加速度为正就是加速，加速度为负就是减速”。必须结合速度方向一起判断。

### 6. 速度、速度变化量、加速度的区别

这三个量很容易混淆：

| 物理量 | 符号 | 描述什么 | 公式或含义 |
| --- | --- | --- | --- |
| 速度 | `v` | 位置变化的快慢和方向 | `v = Δx / Δt` |
| 速度变化量 | `Δv` | 速度改变了多少 | `Δv = v2 - v1` |
| 加速度 | `a` | 速度变化的快慢 | `a = Δv / Δt` |

举例：

一辆车从 `0` 加速到 `20 m/s`，用时 `10 s`。

- 末速度是 `20 m/s`。
- 速度变化量是 `20 m/s`。
- 加速度是 `2 m/s^2`。

如果另一辆车从 `0` 加速到 `20 m/s`，用时 `5 s`：

- 末速度仍是 `20 m/s`。
- 速度变化量仍是 `20 m/s`。
- 加速度变为 `4 m/s^2`。

这说明速度变化量相同，加速度不一定相同，因为还要看所用时间。

### 7. 加速度大不等于速度大

加速度描述的是速度变化快慢，不是速度大小。

所以：

- 速度大，加速度不一定大。
- 速度小，加速度也可能很大。
- 加速度为 0，速度不一定为 0。

例如：

- 匀速飞行的飞机速度可能很大，但如果速度不变，加速度为 `0`。
- 枪膛中刚开始运动的子弹速度可能还不大，但速度增加极快，加速度很大。
- 汽车高速行驶时，如果匀速前进，加速度为 `0`。

判断加速度，不能只看速度大小，要看速度是否变化以及变化得快不快。

### 8. 从 `v-t` 图像看加速度

`v-t` 图像表示速度随时间变化的情况。

在 `v-t` 图像中：

```text
加速度 = 速度变化量 / 时间间隔
```

这正好对应图像的斜率。

所以：

```text
v-t 图像的斜率表示加速度。
```

读图时可以这样判断：

- 图线越陡，加速度大小越大。
- 图线向上倾斜，加速度为正。
- 图线向下倾斜，加速度为负。
- 图线水平，加速度为 0。

如果图线是一条直线，说明加速度不变。

如果图线是曲线，说明加速度在变化；曲线上某一点的切线斜率反映该时刻的加速度。

本节主要掌握直线型 `v-t` 图像的判断。

### 9. 变化率思想

加速度本质上是一种“变化率”。

变化率描述某个量随时间变化得快不快。

例如：

- 速度是位置的变化率。
- 加速度是速度的变化率。
- 水位升高快慢可以看作水位的变化率。
- 商品价格上涨快慢可以看作价格的变化率。

用比值定义物理量，是物理学中非常重要的方法。

本章中有两个典型例子：

```text
速度 = 位移 / 时间
加速度 = 速度变化量 / 时间
```

理解了“变化率”，就能更清楚地区分：

- 速度大不代表加速度大。
- 速度变化量大不代表加速度大。
- 加速度大表示速度变化得快。

## 重点梳理

### 本节核心概念

| 概念 | 含义 | 关键词 |
| --- | --- | --- |
| 加速度 | 描述速度变化快慢的物理量 | 速度变化率 |
| 速度变化量 | 末速度与初速度之差 | `Δv = v2 - v1` |
| 加速运动 | 速度大小增大的运动 | `v` 与 `a` 同向 |
| 减速运动 | 速度大小减小的运动 | `v` 与 `a` 反向 |
| `v-t` 图像斜率 | 表示加速度 | 越陡加速度越大 |

### 本节核心公式

速度变化量：

```text
Δv = v2 - v1
```

加速度：

```text
a = Δv / Δt
```

合并写法：

```text
a = (v2 - v1) / Δt
```

单位：

```text
m/s^2
```

单位含义：

```text
1 m/s^2 表示速度每秒变化 1 m/s
```

### 加速度方向判断

- 加速度方向与速度变化量 `Δv` 的方向相同。
- 直线运动中，`a` 的正负表示方向。
- `a > 0` 表示加速度沿规定正方向。
- `a < 0` 表示加速度沿规定负方向。
- 判断加速还是减速，要比较速度 `v` 和加速度 `a` 的方向。

### 加速与减速判断

| 速度方向 | 加速度方向 | 速度大小变化 | 运动特点 |
| --- | --- | --- | --- |
| 正方向 | 正方向 | 增大 | 加速 |
| 正方向 | 负方向 | 减小 | 减速 |
| 负方向 | 负方向 | 增大 | 加速 |
| 负方向 | 正方向 | 减小 | 减速 |

一句话记忆：

```text
同向加速，反向减速。
```

### `v-t` 图像判断加速度

- 斜率表示加速度。
- 斜率为正，加速度为正。
- 斜率为负，加速度为负。
- 斜率为 0，加速度为 0。
- 斜率绝对值越大，加速度大小越大。

## 难点突破

### 难点 1：速度大，加速度一定大吗？

不一定。

速度描述运动快慢，加速度描述速度变化快慢，二者不是同一个概念。

例如，飞机以 `800 km/h` 匀速飞行：

- 速度很大。
- 但速度不变，所以 `Δv = 0`。
- 加速度为 `0`。

所以速度大不代表加速度大。

判断加速度时要问：

```text
速度有没有变化？变化用了多长时间？
```

### 难点 2：速度变化量大，加速度一定大吗？

不一定。

加速度不仅看速度变化量，还看发生这个变化所用的时间。

例如：

- 物体 A 速度变化 `20 m/s`，用时 `2 s`。
- 物体 B 速度变化 `30 m/s`，用时 `30 s`。

虽然 B 的速度变化量更大，但：

```text
aA = 20 / 2 = 10 m/s^2
aB = 30 / 30 = 1 m/s^2
```

A 的加速度更大。

所以：

```text
加速度 = 速度变化量 / 时间
```

不能只看 `Δv`，也不能只看 `Δt`。

### 难点 3：加速度为负，是不是一定减速？

不一定。

加速度为负只表示加速度方向沿规定负方向，不一定表示速度大小减小。

是否减速，要看速度和加速度方向是否相反。

例如：

- 规定向东为正方向。
- 物体向西运动，`v < 0`。
- 如果 `a < 0`，速度和加速度同向，物体速度大小增大，是加速运动。

所以“负加速度”不等于“减速”。

### 难点 4：加速度方向为什么和速度变化量方向相同？

加速度的定义是：

```text
a = Δv / Δt
```

时间间隔 `Δt` 是正值，不改变方向。

因此加速度方向由 `Δv` 决定。

速度变化量 `Δv` 的方向表示速度“往哪个方向变化”。

例如，规定向右为正方向：

- 速度从 `2 m/s` 增大到 `6 m/s`，`Δv = 4 m/s`，加速度向右。
- 速度从 `6 m/s` 减小到 `2 m/s`，`Δv = -4 m/s`，加速度向左。

注意：加速度方向不是由“末速度方向”单独决定，也不是由“初速度方向”单独决定，而是由速度变化量决定。

### 难点 5：`v-t` 图像斜率和速度大小有什么区别？

在 `v-t` 图像中：

- 某时刻的纵坐标表示速度。
- 图线的斜率表示加速度。

这两个信息不能混淆。

例如，一条水平直线位于 `v = 10 m/s`：

- 速度是 `10 m/s`。
- 斜率是 `0`。
- 加速度是 `0`。

这说明物体速度很大，但加速度为零。

### 难点 6：加速度变小，速度一定变小吗？

不一定。

如果加速度和速度同向，即使加速度变小，速度仍然可能继续增大，只是增大得越来越慢。

例如汽车起步时：

- 一开始加速度较大，速度增加很快。
- 后来加速度变小，速度仍然增加，但增加得慢一些。

所以：

```text
加速度变小，表示速度变化得慢了，不等于速度变小。
```

## 例题讲解

### 例题 1：计算加速度

题目：

一辆汽车从静止开始做直线加速运动，`5 s` 后速度达到 `20 m/s`。求汽车的加速度。

分析：

汽车从静止开始，初速度 `v1 = 0`，末速度 `v2 = 20 m/s`，时间 `Δt = 5 s`。

步骤：

```text
Δv = v2 - v1 = 20 m/s - 0 = 20 m/s
a = Δv / Δt = 20 m/s / 5 s = 4 m/s^2
```

答案：

汽车的加速度为 `4 m/s^2`。

反思：

`4 m/s^2` 表示汽车速度平均每秒增加 `4 m/s`。

### 例题 2：计算减速运动的加速度

题目：

某汽车沿直线向东运动，规定向东为正方向。汽车速度从 `20 m/s` 减小到 `0`，用时 `4 s`。求汽车的加速度，并说明方向。

分析：

向东为正方向，所以初速度为 `20 m/s`，末速度为 `0`。速度变化量为负，说明加速度方向向西。

步骤：

```text
Δv = v2 - v1 = 0 - 20 m/s = -20 m/s
a = Δv / Δt = -20 m/s / 4 s = -5 m/s^2
```

答案：

汽车的加速度为 `-5 m/s^2`，方向向西。

反思：

减速运动中，加速度方向通常与速度方向相反。

### 例题 3：负加速度不一定减速

题目：

规定向东为正方向。某物体向西运动，速度从 `-2 m/s` 变为 `-8 m/s`，用时 `3 s`。求加速度，并判断物体是加速还是减速。

分析：

物体速度为负，说明向西运动。速度大小从 `2 m/s` 变为 `8 m/s`，速度大小增大，所以是加速运动。

步骤：

```text
Δv = v2 - v1 = -8 m/s - (-2 m/s) = -6 m/s
a = Δv / Δt = -6 m/s / 3 s = -2 m/s^2
```

答案：

加速度为 `-2 m/s^2`，方向向西。物体速度和加速度同向，因此做加速运动。

反思：

加速度为负，只表示方向为负方向，不等于减速。

### 例题 4：比较加速度大小

题目：

物体 A 的速度在 `2 s` 内从 `0` 增大到 `12 m/s`；物体 B 的速度在 `6 s` 内从 `0` 增大到 `18 m/s`。哪个物体速度变化更快？

分析：

比较速度变化快慢，就是比较加速度大小。

步骤：

```text
aA = (12 - 0) / 2 = 6 m/s^2
aB = (18 - 0) / 6 = 3 m/s^2
```

答案：

物体 A 的加速度更大，速度变化更快。

反思：

B 的速度变化量更大，但用时也更长，所以加速度反而较小。

### 例题 5：从 `v-t` 图像求加速度

题目：

某物体的 `v-t` 图像是一条直线，`t = 0 s` 时速度为 `2 m/s`，`t = 4 s` 时速度为 `10 m/s`。求物体的加速度。

分析：

`v-t` 图像的斜率表示加速度。可以用两点对应的速度变化量除以时间间隔。

步骤：

```text
Δv = 10 m/s - 2 m/s = 8 m/s
Δt = 4 s - 0 s = 4 s
a = Δv / Δt = 8 m/s / 4 s = 2 m/s^2
```

答案：

物体的加速度为 `2 m/s^2`。

反思：

从 `v-t` 图像求加速度，本质就是求图线斜率。

## 易错点整理

### 易错点 1：把加速度理解为“速度很大”

错误理解：

```text
加速度大，就是速度大。
```

正确理解：

加速度大表示速度变化快，不表示速度本身大。

避免方法：

看到加速度，马上想到：

```text
a = Δv / Δt
```

### 易错点 2：只看速度变化量，不看时间

错误做法：

谁的速度变化量大，就认为谁的加速度大。

正确做法：

加速度取决于速度变化量和发生变化所用时间的比值。

避免方法：

比较加速度时必须计算：

```text
Δv / Δt
```

### 易错点 3：认为加速度为负一定是减速

错误理解：

```text
a < 0，所以物体一定减速。
```

正确理解：

加速度正负只表示方向。是否减速，要看速度和加速度是否反向。

避免方法：

用口诀：

```text
同向加速，反向减速。
```

### 易错点 4：速度变化量顺序写反

错误做法：

```text
Δv = v1 - v2
```

正确做法：

```text
Δv = v2 - v1
```

避免方法：

记住“变化量 = 末量 - 初量”。

### 易错点 5：从 `v-t` 图像读错加速度

错误理解：

图像位置越高，加速度越大。

正确理解：

图像位置高表示速度大；图像斜率大才表示加速度大。

避免方法：

读 `v-t` 图像时分两步：

1. 看纵坐标，读速度。
2. 看斜率，读加速度。

### 易错点 6：忽略单位换算

错误做法：

把 `100 km/h` 直接代入 `a = Δv / Δt`，时间却用秒。

正确做法：

先把速度统一为 `m/s`：

```text
100 km/h = 100 ÷ 3.6 m/s ≈ 27.8 m/s
```

避免方法：

只要公式中时间用 `s`，速度就优先换成 `m/s`。

## 考点考证点整理

### 考点 1：加速度定义式计算

常见问法：

- 给出初速度、末速度和时间，求加速度。
- 给出加速度和时间，求速度变化量。
- 给出加速度和速度变化量，求时间。

解题关键：

```text
a = (v2 - v1) / Δt
```

先统一单位，再代入公式。

### 考点 2：加速度方向判断

常见问法：

- 判断加速度方向。
- 根据正负号说明运动情况。
- 判断物体是加速还是减速。

解题关键：

- 加速度方向与 `Δv` 方向相同。
- `Δv = v2 - v1`。
- 判断加速减速要比较 `v` 与 `a` 的方向。

### 考点 3：速度、速度变化量、加速度辨析

常见问法：

- 速度大，加速度是否一定大？
- 速度变化量大，加速度是否一定大？
- 加速度为 0，速度是否一定为 0？

解题关键：

把三个问题分开：

- 速度：运动快慢。
- 速度变化量：速度改变多少。
- 加速度：速度改变快慢。

### 考点 4：从 `v-t` 图像求加速度

常见问法：

- 根据图像两点求加速度。
- 比较多条图线的加速度大小。
- 判断加速度方向。

解题关键：

```text
v-t 图像斜率 = 加速度
```

斜率越大，加速度越大；斜率为负，加速度为负。

### 考点 5：实际情境中的加速度

常见问法：

- 汽车起步加速度。
- 汽车刹车加速度。
- 交通工具性能比较。
- 子弹、飞机、列车等实例判断。

解题关键：

把生活语言翻译成物理量：

- “提速快”对应加速度大。
- “刹车猛”对应加速度大小大，方向与速度相反。
- “匀速”对应加速度为 0。

## 练习题

### 基础题

1. 某物体速度从 `0` 增大到 `12 m/s`，用时 `4 s`。求加速度。

2. 某汽车速度从 `20 m/s` 减小到 `5 m/s`，用时 `3 s`。规定汽车运动方向为正方向，求加速度。

3. 判断下列说法是否正确，并说明理由：加速度越大，物体速度一定越大。

4. 加速度的单位 `m/s^2` 表示什么含义？

5. 在 `v-t` 图像中，图线水平表示什么？

### 巩固题

6. 规定向右为正方向，某物体速度从 `-4 m/s` 变为 `-10 m/s`，用时 `2 s`。求加速度，并判断物体是加速还是减速。

7. 小汽车从静止加速到 `100 km/h` 用时 `10 s`。求它的加速度大小。结果保留两位小数。

8. 甲物体速度在 `5 s` 内变化 `20 m/s`，乙物体速度在 `2 s` 内变化 `12 m/s`。哪个物体速度变化更快？

9. 某物体沿直线运动，速度为正，加速度为负。它一定向反方向运动吗？它的速度大小如何变化？

10. 某 `v-t` 图像是一条向下倾斜的直线，这说明物体加速度有什么特点？

### 提升题

11. 规定向东为正方向，某物体初速度为 `12 m/s`，经过 `4 s` 后速度变为 `-4 m/s`。求加速度，并说明运动方向是否发生改变。

12. A、B 两辆车都从静止开始加速。A 车 `8 s` 后达到 `24 m/s`，B 车 `5 s` 后达到 `20 m/s`。比较两车加速度大小。

13. 某汽车向东行驶，速度大小为 `20 m/s`，然后开始刹车，`2 min` 后停止。规定向东为正方向，求加速度大小和方向。

14. 某物体的 `v-t` 图像经过两点 `(0 s, 6 m/s)` 和 `(3 s, 0 m/s)`。求加速度，并判断物体在这 `3 s` 内做加速运动还是减速运动。

15. 举出一个“速度很大但加速度为 0”的例子，再举出一个“速度很小但加速度很大”的例子，并说明理由。

## 练习题答案

### 基础题答案

1. 加速度：

```text
a = Δv / Δt = (12 - 0) m/s / 4 s = 3 m/s^2
```

答案：加速度为 `3 m/s^2`。

2. 速度变化量：

```text
Δv = v2 - v1 = 5 m/s - 20 m/s = -15 m/s
```

加速度：

```text
a = Δv / Δt = -15 m/s / 3 s = -5 m/s^2
```

答案：加速度为 `-5 m/s^2`，方向与汽车运动方向相反。

3. 说法错误。

加速度描述速度变化快慢，不描述速度大小。物体速度很大时，如果速度不变，加速度可以为 `0`。

4. `m/s^2` 表示速度变化快慢的单位。`1 m/s^2` 表示速度每秒变化 `1 m/s`。

5. `v-t` 图像中图线水平，表示速度不随时间变化，加速度为 `0`。

### 巩固题答案

6. 速度变化量：

```text
Δv = v2 - v1 = -10 m/s - (-4 m/s) = -6 m/s
```

加速度：

```text
a = Δv / Δt = -6 m/s / 2 s = -3 m/s^2
```

物体速度为负，加速度也为负，二者同向，所以速度大小增大，是加速运动。

答案：加速度为 `-3 m/s^2`，物体做加速运动。

7. 先换单位：

```text
100 km/h = 100 ÷ 3.6 m/s ≈ 27.78 m/s
```

加速度：

```text
a = (27.78 - 0) m/s / 10 s = 2.78 m/s^2
```

答案：加速度大小约为 `2.78 m/s^2`。

8. 甲的加速度大小：

```text
a甲 = 20 / 5 = 4 m/s^2
```

乙的加速度大小：

```text
a乙 = 12 / 2 = 6 m/s^2
```

答案：乙物体速度变化更快。

9. 不一定向反方向运动。

速度为正，说明物体仍沿正方向运动；加速度为负，说明加速度方向与速度方向相反。因此物体速度大小减小，正在减速，但只要速度还没有变为负，就没有向反方向运动。

10. `v-t` 图像向下倾斜，说明斜率为负，因此加速度为负。如果图线是直线，说明加速度恒定且为负。

### 提升题答案

11. 加速度：

```text
Δv = v2 - v1 = -4 m/s - 12 m/s = -16 m/s
a = Δv / Δt = -16 m/s / 4 s = -4 m/s^2
```

答案：加速度为 `-4 m/s^2`，方向向西。

运动方向发生了改变，因为速度从正值变成负值，说明物体先沿向东方向运动，后来沿向西方向运动。

12. A 车加速度：

```text
aA = 24 m/s / 8 s = 3 m/s^2
```

B 车加速度：

```text
aB = 20 m/s / 5 s = 4 m/s^2
```

答案：B 车加速度更大。

13. `2 min = 120 s`。

规定向东为正方向：

```text
v1 = 20 m/s
v2 = 0
Δt = 120 s
```

加速度：

```text
a = (0 - 20) m/s / 120 s = -1/6 m/s^2 ≈ -0.17 m/s^2
```

答案：加速度大小约为 `0.17 m/s^2`，方向向西，即与汽车运动方向相反。

14. 加速度等于 `v-t` 图像斜率：

```text
a = (0 - 6) m/s / (3 - 0) s = -2 m/s^2
```

答案：加速度为 `-2 m/s^2`。物体初速度为正，加速度为负，速度大小减小，所以在这 `3 s` 内做减速运动。

15. 示例一：

高空中匀速飞行的飞机速度很大，但如果速度大小和方向都不变，则加速度为 `0`。

示例二：

枪膛中刚开始运动的子弹速度可能还很小，但在火药燃气作用下速度迅速增大，因此加速度很大。

这说明速度大小和加速度大小没有必然对应关系，加速度描述的是速度变化快慢。
