# 4.4 力学单位制

## 本节学习目标

- 理解基本量、基本单位、导出量、导出单位。
- 知道力学中的基本单位是 `m`、`kg`、`s`。
- 会进行常见单位换算。
- 会用国际单位制进行动力学计算。
- 能用单位关系检查公式是否明显错误。

## 核心知识点讲解

### 1. 基本量和基本单位

在力学中，选作基础的物理量是：

- 长度。
- 质量。
- 时间。

对应国际单位制基本单位：

```text
m、kg、s
```

### 2. 导出量和导出单位

由基本量通过物理关系推导出来的量叫导出量。

例如：

```text
v = Δx/Δt
```

速度单位是：

```text
m/s
```

加速度：

```text
a = Δv/Δt
```

单位是：

```text
m/s^2
```

力：

```text
F = ma
```

单位是：

```text
N = kg·m/s^2
```

### 3. 国际单位制

国际单位制简称 SI。力学中常用：

| 物理量 | 单位 |
| --- | --- |
| 长度 | `m` |
| 质量 | `kg` |
| 时间 | `s` |
| 速度 | `m/s` |
| 加速度 | `m/s^2` |
| 力 | `N` |

### 4. 常见单位换算

```text
1 t = 1000 kg
1 g = 0.001 kg
1 km/h = 1/3.6 m/s
1 N = 1 kg·m/s^2
```

使用 `F = ma` 时，质量必须用 `kg`。

### 5. 用单位检查公式

公式左右两边单位必须一致。

例如体积单位应为 `m^3`。若某公式右边单位是 `m^4`，它不可能是体积公式。

## 重点梳理

- 力学基本单位：`m`、`kg`、`s`。
- `1 N = 1 kg·m/s^2`。
- `1 J = 1 N·m = 1 kg·m^2/s^2`。
- 代入公式前先统一单位。

## 难点突破

### 难点 1：`N/kg` 和 `m/s^2` 为什么等价

因为：

```text
1 N/kg = (1 kg·m/s^2)/kg = 1 m/s^2
```

所以 `g` 的单位可写作 `N/kg` 或 `m/s^2`。

### 难点 2：单位一致公式一定正确吗

不一定。单位一致只是必要条件，不是充分条件。

## 例题讲解

### 例题 1：单位换算

`700 g = 0.7 kg`，物体受 `1.4 N` 力：

```text
a = F/m = 1.4/0.7 = 2 m/s^2
```

### 例题 2：速度和位移

从静止开始，`a = 2 m/s^2`，`t = 5 s`：

```text
v = at = 10 m/s
x = 1/2 at^2 = 25 m
```

### 例题 3：焦耳单位

```text
W = Fl
J = N·m = kg·m^2/s^2
```

## 易错点整理

- 克未换千克。
- 吨未换千克。
- 千米每时未换米每秒。
- 结果不写单位。
- 单位检查时忽略平方或立方。

## 考点考证点整理

- `N`、`J` 的推导。
- 单位换算。
- 单位检查公式。

## 练习题

### 基础题

1. 力学中的三个基本单位是什么？
2. `1 N` 等于什么？
3. `72 km/h` 等于多少 `m/s`？
4. `2 t` 等于多少 `kg`？
5. `500 g` 等于多少 `kg`？

### 巩固题

6. `300 g` 物体受 `0.6 N` 力，求加速度。
7. 推导 `J` 与基本单位关系。
8. 判断 `V = 1/3πR^3h` 是否可能是体积公式。

### 提升题

9. 汽车质量 `2 t`，受制动力 `1.2×10^4 N`，求加速度大小。
10. 飞船受 `900 N` 推力，`3 s` 内速度增加 `0.9 m/s`，求质量。

## 练习题答案

1. `m`、`kg`、`s`。
2. `1 kg·m/s^2`。
3. `20 m/s`。
4. `2000 kg`。
5. `0.5 kg`。
6. `m=0.3 kg`，`a=0.6/0.3=2 m/s^2`。
7. `J=N·m=kg·m^2/s^2`。
8. 不可能，右边单位是 `m^4`。
9. `a=1.2×10^4/2000=6 m/s^2`。
10. `a=0.9/3=0.3 m/s^2`，`m=900/0.3=3000 kg`。
