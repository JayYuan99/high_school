# 4.6 超重和失重

## 本节学习目标

- 理解体重计示数反映支持力，不一定等于重力。
- 掌握超重、失重、完全失重的概念。
- 会根据加速度方向判断超重或失重。
- 会计算电梯中的支持力或压力。
- 能解释下蹲、站起、电梯运动、自由下落水瓶等现象。

## 核心知识点讲解

### 1. 体重计示数是什么

人站在体重计上，体重计示数反映人对体重计的压力大小。

根据牛顿第三定律，它等于体重计对人的支持力大小。

所以体重计示数本质上是：

```text
FN
```

不一定等于 `mg`。

### 2. 超重

物体对支持物的压力或对悬挂物的拉力大于重力的现象叫超重。

加速度向上时：

```text
FN - mg = ma
FN = m(g+a) > mg
```

### 3. 失重

物体对支持物的压力或对悬挂物的拉力小于重力的现象叫失重。

加速度向下时：

```text
mg - FN = ma
FN = m(g-a) < mg
```

### 4. 完全失重

若向下加速度：

```text
a = g
```

则：

```text
FN = 0
```

物体处于完全失重状态。

### 5. 判断关键

判断超重失重看加速度方向，不看速度方向。

- 加速度向上：超重。
- 加速度向下：失重。
- 加速度为 0：正常。

### 6. 下蹲和站起

下蹲：

- 开始向下加速：失重。
- 后来向下减速：超重。
- 静止：正常。

站起：

- 开始向上加速：超重。
- 后来向上减速：失重。
- 静止：正常。

## 重点梳理

```text
加速度向上：FN = m(g+a)，超重
加速度向下：FN = m(g-a)，失重
a = g 向下：FN = 0，完全失重
```

超重、失重不是重力变了，而是支持力或拉力变了。

## 难点突破

### 难点 1：向上运动一定超重吗

不一定。向上减速时加速度向下，是失重。

### 难点 2：向下运动一定失重吗

不一定。向下减速时加速度向上，是超重。

### 难点 3：完全失重是不是没有重力

不是。完全失重时仍受重力，只是没有支持力或拉力。

## 例题讲解

### 例题 1：电梯加速上升

质量 `60 kg`，电梯以 `0.25 m/s^2` 加速上升，取 `g=9.8 m/s^2`：

```text
FN = m(g+a)=60(9.8+0.25)=603 N
```

人对电梯压力大小为 `603 N`。

### 例题 2：电梯加速下降

质量 `50 kg`，电梯以 `2 m/s^2` 加速下降，取 `g=10 m/s^2`：

```text
FN = m(g-a)=50(10-2)=400 N
```

### 例题 3：火箭超重

某器官平时重 `10 N`，火箭加速度 `3.5g` 向上：

```text
FN = m(g+3.5g)=4.5mg=45 N
```

## 易错点整理

- 超重不是重力变大。
- 失重不是重力消失。
- 判断超重失重看加速度，不看速度。
- 匀速上升或下降都不超重不失重。
- 人对电梯压力与电梯对人的支持力大小相等。

## 考点考证点整理

- 电梯启动、制动过程判断。
- `FN = m(g±a)` 计算。
- 完全失重现象。
- 下蹲、站起图像分析。
- 航天器内失重现象解释。

## 练习题

### 基础题

1. 体重计示数反映什么？
2. 什么是超重？
3. 什么是失重？
4. 什么是完全失重？
5. 判断超重失重看什么方向？

### 巩固题

6. `60 kg` 人随电梯以 `1 m/s^2` 加速上升，取 `g=10 m/s^2`，求示数。
7. `60 kg` 人随电梯以 `1 m/s^2` 加速下降，求示数。
8. 电梯匀速上升时是否超重？

### 提升题

9. 火箭向上加速度 `3g`，物体平时重 `100 N`，求支持力。
10. `50 kg` 人在加速下降电梯中体重计示数 `400 N`，取 `g=10 m/s^2`，求加速度。

## 练习题答案

1. 支持力或压力的大小。
2. 压力或拉力大于重力的现象。
3. 压力或拉力小于重力的现象。
4. 压力或拉力为 0 的失重状态。
5. 加速度方向。
6. `FN=60(10+1)=660 N`。
7. `FN=60(10-1)=540 N`。
8. 不，匀速时加速度为 0。
9. `FN=4mg=400 N`。
10. `400=50(10-a)`，`a=2 m/s^2`，方向向下。
