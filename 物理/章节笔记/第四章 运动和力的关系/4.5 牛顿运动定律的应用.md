# 4.5 牛顿运动定律的应用

## 本节学习目标

- 理解动力学问题的两条基本思路。
- 会从受力情况确定运动情况。
- 会从运动情况确定受力情况。
- 能把 `F合 = ma` 与匀变速直线运动公式结合。
- 会处理水平面、斜面、摩擦、分段运动等问题。

## 核心知识点讲解

### 1. 两类基本问题

牛顿第二定律把力和运动联系起来。

第一类：

```text
已知受力 -> 求加速度 -> 求运动情况
```

第二类：

```text
已知运动情况 -> 求加速度 -> 求受力
```

加速度是连接受力和运动的桥梁。

### 2. 从受力确定运动

步骤：

1. 选研究对象。
2. 受力分析。
3. 求合力。
4. 由 `F合 = ma` 求加速度。
5. 用运动学公式求速度、位移、时间。

### 3. 从运动确定受力

步骤：

1. 根据运动学公式求加速度。
2. 对物体受力分析。
3. 根据 `F合 = ma` 求未知力。

### 4. 水平面滑动摩擦

若水平面上滑动且 `FN = mg`：

```text
Ff = μmg
a = -μg
```

摩擦力使物体减速。

### 5. 斜面问题

常沿斜面和垂直斜面建轴：

```text
mg sinθ
mg cosθ
```

沿斜面列动力学方程，垂直斜面方向通常平衡。

### 6. 分段运动

不同阶段受力不同，加速度不同，要分段处理。

前一段末速度通常是后一段初速度。

## 重点梳理

```text
受力 -> F合 -> a -> 运动
运动 -> a -> F合 -> 受力
```

常用公式：

```text
F合 = ma
v = v0 + at
x = v0t + 1/2 at^2
v^2 - v0^2 = 2ax
```

## 难点突破

### 难点 1：为什么加速度是桥梁

牛顿第二定律由力求加速度，运动学由加速度求速度和位移。

### 难点 2：为什么要分段

匀变速公式只适用于加速度不变的阶段。受力变了，加速度变了，必须另起一段。

### 难点 3：方向符号

先规定正方向，减速时加速度通常为负。

## 例题讲解

### 例题 1：冰壶滑行

冰壶初速度 `3.4 m/s`，`μ=0.02`，`g=10 m/s^2`：

```text
a = -μg = -0.2 m/s^2
0 - 3.4^2 = 2×(-0.2)x
x = 28.9 m
```

### 例题 2：由运动求阻力

滑雪者质量 `75 kg`，沿 `30°` 斜面下滑，`v0=2 m/s`，`t=5 s`，`x=60 m`。

```text
a = 2(x-v0t)/t^2 = 2(60-10)/25 = 4 m/s^2
Ff = m(g sin30° - a) = 75(5-4)=75 N
```

### 例题 3：两个力产生运动

质量 `20 kg`，两个 `50 N` 水平力夹角 `60°`：

```text
F合 = 2×50×cos30° = 86.6 N
a = 86.6/20 = 4.33 m/s^2
```

## 易错点整理

- 没有画受力图就列式。
- 把单个力当合力。
- 斜面上误写 `FN = mg`。
- 分段问题混用一个加速度。
- 忽略方向正负。

## 考点考证点整理

- 从受力求运动。
- 从运动求受力。
- 水平面摩擦减速。
- 斜面动力学。
- 分段运动。

4. 水平面滑动摩擦减速时 `a` 如何表示？
5. 为什么分段运动要分段求解？

### 巩固题

6. `10 kg` 物体受合力 `20 N`，从静止运动 `4 s`，求速度。
7. 初速度 `10 m/s`，加速度 `-2 m/s^2`，求停止距离。
8. `5 kg` 物体受 `30 N` 拉力和 `10 N` 摩擦力，从静止运动 `3 s`，求位移。

### 提升题

10. 冰壶初速度 `3.4 m/s`，先在 `μ=0.02` 冰面滑行 `10 m`，再进入 `μ=0.018` 冰面，求第二段距离。

## 练习题答案

1. 加速度。
2. 受力分析 -> 求合力 -> 求加速度 -> 用运动学公式。
3. 运动学求加速度 -> 受力分析 -> 牛顿第二定律求力。
4. `a = -μg`。
5. 因为不同阶段受力和加速度可能不同。
6. `a=20/10=2 m/s^2`，`v=at=8 m/s`。
7. `0-10^2=2×(-2)x`，`x=25 m`。
8. `F合=20 N`，`a=4 m/s^2`，`x=1/2×4×9=18 m`。
9. `54 km/h=15 m/s`，`a=-225/(2×17.2)≈-6.54 m/s^2`，`μ≈0.654`。
10. 第一段后 `v1^2=3.4^2+2×(-0.2)×10=7.56`；第二段 `a=-0.18`，`x=7.56/(0.36)=21 m`。
