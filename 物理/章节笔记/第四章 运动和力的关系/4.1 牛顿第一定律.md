# 4.1 牛顿第一定律

## 本节学习目标

学完本节，需要能做到：

- 说清亚里士多德经验观点为什么会把人引向“力维持运动”的错误认识。
- 理解伽利略理想斜面实验的实验事实、逻辑推理和科学方法价值。
- 准确表述牛顿第一定律，知道它也叫惯性定律。
- 理解“运动状态改变”指速度大小或方向改变。
- 理解惯性是一切物体固有的属性，质量是惯性大小的量度。
- 会用惯性解释安全带、飞机投弹、跳起后落回原地等教材情境。
- 初步知道惯性参考系和非惯性参考系的区别。

## 核心知识点讲解

### 一、知识对象与物理情境

本节讨论的是“运动和力到底是什么关系”。日常经验容易让人形成这样的判断：推车时车会动，不推车时车会停，所以似乎“物体运动必须有力维持”。

教材提醒我们，这种明显可见的线索可能引出错误判断。滑冰运动员不用力后慢慢停下，并不是因为“没有力就不能运动”，而是因为冰面摩擦力和空气阻力在改变运动状态。

因此，本节的核心问题是：

$$
\text{力是维持运动的原因，还是改变运动状态的原因？}
$$

### 二、核心概念与物理意义

**运动状态**由速度描述。速度大小改变，或速度方向改变，都叫运动状态发生改变。

**惯性**是物体保持原来静止状态或匀速直线运动状态的性质。惯性不是一种力，不能说“受到惯性”“惯性力使物体运动”。惯性是物体固有的属性，一切物体都有惯性。

**质量**不仅表示物体所含物质的多少，也可以从惯性角度理解为描述惯性大小的物理量。质量越大，物体维持原有运动状态的能力越强，运动状态越难改变。

### 三、关键规律、公式与适用条件

牛顿第一定律的内容是：

> 一切物体总保持匀速直线运动状态或静止状态，除非作用在它上面的力迫使它改变这种状态。

它揭示的运动和力关系是：

- 力不是维持物体运动状态的原因。
- 力是改变物体运动状态的原因。
- 如果物体不受力或所受合力为零，它将保持静止或匀速直线运动状态。

牛顿第一定律描述的是一种理想状态。现实中物体总与周围物体有相互作用，完全不受力的物体不存在，所以牛顿第一定律不能靠一个真实实验直接验证，而是在实验事实基础上通过理想化推理得到。

### 四、典型模型与分析方法

伽利略理想斜面实验是本节最重要的方法线索。

实验与推理过程：

1. 小球沿一个斜面从静止滚下，再冲上另一个斜面。
2. 如果没有摩擦，小球会到达原来的高度。
3. 第二个斜面倾角减小时，小球仍要到达原来的高度，但运动距离更长。
4. 当第二个斜面变成水平面时，小球为了达到原高度将永远运动下去。
5. 结论：如果没有阻力，运动物体将保持匀速直线运动；力不是维持运动的原因。

### 六、题型应用与迁移

解本节题目时通常按下面路径思考：

1. 判断题目是在问“力和运动关系”，还是问“惯性现象”。
2. 如果物体停下或减速，先找阻力，不要说“没有力所以停下”。
3. 如果解释安全带、投弹、跳起落回原地，先说明物体原来的运动状态，再说明惯性使它保持这种状态。
4. 如果讨论惯性大小，只看质量，不看速度、受力或运动状态。
5. 如果出现加速参考系，要注意牛顿第一定律只在惯性参考系中成立。

## 重点梳理

- 亚里士多德观点的错误在于把阻力作用下的经验现象误认为“运动需要力维持”。
- 伽利略理想斜面实验说明：没有阻力时，运动物体会保持匀速直线运动。
- 牛顿第一定律揭示：力是改变运动状态的原因，不是维持运动的原因。
- 惯性是一切物体固有的属性，不是力。
- 质量是惯性大小的量度，质量越大，惯性越大。
- 牛顿第一定律只在惯性参考系中成立；加速参考系通常是非惯性参考系。

## 难点突破

### 难点 1：滑冰运动员不用力后停下，是否与牛顿第一定律矛盾？

不矛盾。牛顿第一定律说的是不受力或合力为零时物体保持匀速直线运动。滑冰运动员慢慢停下，是因为受到冰面摩擦力和空气阻力，合力不为零，运动状态被改变。

### 难点 2：为什么牛顿第一定律不能直接用实验验证？

现实中不存在完全不受力的物体，也不可能把所有阻力彻底消除。牛顿第一定律是基于实验事实，再经过理想化推理得到的规律。

### 难点 3：速度越大，惯性越大吗？

不是。惯性大小只由质量决定。高速汽车更难停下，是因为速度变化量大，刹车过程需要更长时间或更大作用力，不是因为惯性随速度变大。

### 难点 4：加速纸面参考系中为什么会出现“矛盾”？

以加速纸面为参考系，小球相对纸面运动状态改变，但水平方向几乎不受力，看起来和牛顿第一定律矛盾。换成地面参考系，小球近似保持静止，矛盾消失。这说明牛顿第一定律是否成立与参考系有关。

## 例题讲解

### 例题 1：解释安全带

题目：汽车急刹车时，乘客为什么会向前倾？为什么要系安全带？

分析：急刹车时汽车速度迅速减小，乘客原来随车向前运动。由于惯性，乘客要保持原来的向前运动状态。

步骤：

1. 研究对象：乘客。
2. 原运动状态：随汽车向前运动。
3. 急刹车后：汽车减速，乘客因惯性继续向前。
4. 安全带作用：给乘客向后的力，使乘客随车减速。

答案：乘客向前倾是惯性现象。安全带给乘客向后的作用力，改变乘客运动状态，减少伤害。

### 例题 2：飞机投弹

题目：飞机水平飞行时，如果目标在飞机正下方才投下炸弹，能击中目标吗？

分析：炸弹离开飞机时具有与飞机相同的水平速度，离开后仍因惯性保持水平运动，同时在重力作用下下落。

答案：不能。炸弹会落在目标前方。若要击中目标，应在目标到达飞机正下方之前投弹。

### 例题 3：判断惯性说法

题目：“汽车速度越大，刹车后越难停下来，说明速度越大惯性越大。”这句话对吗？

答案：不对。惯性只由质量决定。速度越大时需要改变的速度量更大，所以更难停下，但惯性没有因为速度增大而增大。

## 易错点整理

- 错误：力维持运动。正确：力改变运动状态。
- 错误：没有力物体就会停。正确：没有力或合力为零时，运动物体保持匀速直线运动。
- 错误：惯性是一种力。正确：惯性是物体属性。
- 错误：运动物体才有惯性。正确：一切物体都有惯性。
- 错误：速度越大惯性越大。正确：惯性只与质量有关。
- 错误：牛顿第一定律在任何参考系都成立。正确：它只在惯性参考系中成立。

## 考点考证点整理

### 考点 1：力和运动关系辨析

- 出题思路：常用推车、滑冰、抛体、刹车等生活情境，要求判断“力是否维持运动”。
- 关键条件：看物体是否受到阻力或合力是否为零，分清“保持运动状态”和“改变运动状态”。
- 解答要点：写出“力不是维持运动的原因，力是改变运动状态的原因”；停下或减速是阻力作用的结果。
- 易扣分点：写成“物体没有力就不能运动”或只说“因为惯性”而不说明阻力改变运动状态。

### 考点 2：伽利略理想斜面实验

- 出题思路：给出斜面实验过程或频闪图，要求补全推理或说明科学方法。
- 关键条件：无摩擦、回到原高度、第二斜面倾角逐渐减小、水平面极限情形。
- 解答要点：按“回到原高度 -> 斜面越缓距离越长 -> 水平时一直运动”组织答案，并指出理想实验抓住主要因素、忽略摩擦。
- 易扣分点：把理想实验当作完全真实可直接完成的实验，或漏写“无摩擦”条件。

### 考点 3：牛顿第一定律内容和惯性

- 出题思路：考查定律原文、惯性概念、运动状态改变的含义。
- 关键条件：静止、匀速直线运动、速度大小改变、速度方向改变、质量。
- 解答要点：定律表述要包含“总保持静止或匀速直线运动状态”和“除非力迫使它改变”；惯性是一切物体固有属性，质量量度惯性。
- 易扣分点：把惯性说成力，把惯性大小和速度、受力大小联系起来。

### 考点 4：惯性现象解释

- 出题思路：常给安全带、投弹、跳起落回原地、急刹车前倾等情境。
- 关键条件：物体原来的运动状态、外力如何改变运动状态、研究对象是谁。
- 解答要点：先写原运动状态，再写由于惯性保持原状态，最后结合外力说明结果。
- 易扣分点：只写“因为惯性”四个字，没有说明保持什么运动状态。

### 考点 5：惯性参考系

- 出题思路：以拉纸和小钢球、加速车厢等情境考查牛顿第一定律的适用参考系。
- 关键条件：参考系是否加速、物体水平方向是否受力、相对哪个参考系观察。
- 解答要点：在惯性参考系中，不受力物体保持静止或匀速直线运动；加速参考系中牛顿第一定律不直接成立。
- 易扣分点：忽略参考系，或把非惯性系中的现象直接套用牛顿第一定律。

## 练习题

### 教材练习与应用

1. 教材练习与应用第 1 题：回答下列问题。
   1. 飞机投弹时，如果当目标在飞机的正下方时投下炸弹，能击中目标吗？为什么？
   2. 地球由西向东自转，向上跳起来以后，为什么还落在原地附近，而不落到原地的西边？
   3. 我国道路交通安全法规定，在各种小型车辆里乘坐的人必须系好安全带。为什么要有这样的规定？
   4. 一位同学说，向上抛出的物体在空中向上运动时，肯定受到了向上的作用力，否则它不可能向上运动。这个结论错在哪里？
2. 教材练习与应用第 2 题：伽利略在理想斜面实验中提出：如果另一个斜面的倾角减小至 $0^\circ$，小球为达到原来的高度，将永远运动下去。请说明他得到这个结论的理由。
3. 教材练习与应用第 3 题：判断下列关于惯性的说法哪些正确、哪些错误，并说明理由。
   1. 汽车速度越大，刹车后越难停下来，表明物体速度越大，惯性越大。
   2. 汽车转弯后前进方向发生改变，表明速度方向改变，其惯性也随之改变。
   3. 被抛出的小球，尽管速度大小和方向都改变了，但惯性不变。
   4. 要使速度相同的沙袋在相同时间内停下来，对大沙袋用力比对小沙袋用力大，表明质量大的物体惯性大。

### 巩固题

4. 滑冰运动员停止蹬冰后会逐渐停下，这是否说明“力是维持运动的原因”？请解释。
5. 迅速向右拉动桌面上的纸，小球相对纸面向左运动。分别以纸面和地面为参考系说明现象。

## 练习题答案

1. 教材练习与应用第 1 题答案：
   1. 不能。炸弹离开飞机后仍具有向前的水平速度，同时在重力作用下下落，会落在目标前方。
   2. 人跳起前已经随地球一起由西向东运动，跳起后由于惯性仍保持原来的水平运动状态，所以落回原地附近。
   3. 急刹车或碰撞时，乘客由于惯性要保持原来的运动状态，安全带能给乘客作用力，使其随车减速，减少伤害。
   4. 错在认为向上运动必须受向上的力。物体离手后仍向上运动是由于惯性；若忽略空气阻力，物体只受重力，重力使它速度逐渐减小。
2. 教材练习与应用第 2 题答案：在无摩擦条件下，小球从一侧斜面滚下后会冲上另一侧斜面并达到原高度；当另一侧斜面倾角减小时，为达到原高度，小球运动距离变长；当倾角减小到 $0^\circ$ 成为水平面时，小球无法通过升高来达到原高度，只能一直运动下去。这说明没有阻力时，运动物体会保持匀速直线运动。
3. 教材练习与应用第 3 题答案：
   1. 错。惯性只由质量决定，速度大只是需要改变的速度量更大。
   2. 错。速度方向改变不说明惯性改变，质量不变则惯性不变。
   3. 对。小球质量不变，惯性不变。
   4. 对。相同时间内使速度相同的大沙袋停下需要更大的力，说明质量大的物体惯性大。
4. 不说明。运动员停下是因为受到冰面摩擦力和空气阻力，这些力改变了运动员的运动状态。
5. 以纸面为参考系，小球相对纸面向左运动；以地面为参考系，小球水平方向几乎不受力，近似保持原来的静止状态。纸面是加速参考系，牛顿第一定律不直接适用；地面可近似看作惯性参考系。
