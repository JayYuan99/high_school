# 4.3 牛顿第二定律

## 本节学习目标

- 掌握牛顿第二定律的内容和表达式。
- 理解 `F = ma` 中的 `F` 是合力。
- 知道加速度方向与合力方向相同。
- 会用牛顿第二定律求合力、质量或加速度。
- 能把牛顿第二定律与运动学公式结合解题。

## 核心知识点讲解

### 1. 牛顿第二定律内容

物体加速度的大小跟它受到的合力成正比，跟它的质量成反比；加速度方向与合力方向相同。

表达式：

```text
F合 = ma
```

其中：

- `F合`：物体所受合力，单位 `N`。
- `m`：物体质量，单位 `kg`。
- `a`：物体加速度，单位 `m/s^2`。

### 2. `F` 必须是合力

实际物体可能受到多个力，加速度由所有力的合力决定。

例如汽车起步：

```text
F合 = F牵引 - F阻力
```

不能只用牵引力计算加速度。

### 3. 加速度方向

加速度方向与合力方向相同，不一定与速度方向相同。

- 合力与速度同向：加速。
- 合力与速度反向：减速。
- 合力与速度成角度：速度方向改变。

### 4. 力的单位

在国际单位制中：

```text
1 N = 1 kg·m/s^2
```

即质量为 `1 kg` 的物体获得 `1 m/s^2` 加速度所需的合力为 `1 N`。

### 5. 质量与惯性

在相同合力作用下：

```text
a = F/m
```

质量越大，加速度越小，运动状态越难改变。因此质量是惯性大小的量度。

## 重点梳理

```text
F合 = ma
```

- 合力决定加速度。
- 加速度方向与合力方向一致。
- 合力为 0 时，加速度为 0。
- 使用公式必须统一国际单位制。

## 难点突破

### 难点 1：很小的力是否一定产生加速度

只有合力不为 0 才产生加速度。提重箱子提不动，是因为箱子所受合力仍为 0。

### 难点 2：速度方向和加速度方向

速度表示物体正在怎样运动；加速度表示速度怎样变化。二者可以不同向。

### 难点 3：单位必须统一

质量用 `kg`，加速度用 `m/s^2`，力才是 `N`。

## 例题讲解

### 例题 1：由合力求加速度

质量 `4 kg` 的物体受到 `12 N` 合力：

```text
a = F/m = 12/4 = 3 m/s^2
```

### 例题 2：汽车阻力

质量 `1000 kg` 的汽车以 `20 m/s` 行驶，取消动力后 `50 s` 停下：

```text
a = (0-20)/50 = -0.4 m/s^2
F = ma = 1000×(-0.4) = -400 N
```

阻力大小 `400 N`，方向与运动方向相反。

### 例题 3：悬线测加速度

列车加速时，小球悬线与竖直方向夹角为 `θ`，则：

```text
a = g tanθ
```

## 易错点整理

- 把某一个力当成合力。
- 根据速度方向判断加速度方向。
- 忘记单位换算。
- 物体静止就认为不受力。
- 忽略阻力。

## 考点考证点整理

- 牛顿第二定律内容和公式。
- 合力与加速度方向。
- `1 N = 1 kg·m/s^2`。
- 水平面、斜面、悬线小球综合题。
- 由运动情况求受力。

## 练习题

### 基础题

1. 牛顿第二定律公式是什么？
2. 公式中的 `F` 指什么？
3. 加速度方向由什么决定？
4. `1 N` 等于什么？
5. 质量越大惯性越大还是越小？

### 巩固题

6. `2 kg` 物体受合力 `10 N`，求加速度。
7. 物体受合力 `6 N` 时加速度 `2 m/s^2`，求质量。
8. `3 kg` 物体加速度 `4 m/s^2`，求合力。

### 提升题

9. 水平面上物体受 `20 N` 拉力和 `5 N` 阻力，质量 `5 kg`，求加速度。
10. `30 kg` 手推车受 `60 N` 推力时加速度 `1.5 m/s^2`，求阻力。

## 练习题答案

1. `F合 = ma`。
2. 物体所受合力。
3. 合力方向。
4. `1 kg·m/s^2`。
5. 越大。
6. `a = 10/2 = 5 m/s^2`。
7. `m = 6/2 = 3 kg`。
8. `F = 3×4 = 12 N`。
9. `F合 = 20-5 = 15 N`，`a = 15/5 = 3 m/s^2`。
10. `F合 = ma = 30×1.5 = 45 N`，阻力 `60-45 = 15 N`。
