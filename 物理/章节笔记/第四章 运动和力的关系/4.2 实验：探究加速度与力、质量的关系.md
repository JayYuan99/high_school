# 4.2 实验：探究加速度与力、质量的关系

## 本节学习目标

- 知道本实验要探究 `a`、`F`、`m` 三者关系。
- 掌握控制变量法。
- 会分析 `a-F` 图像和 `a-1/m` 图像。
- 理解平衡阻力、测量加速度、槽码质量远小于小车质量等实验要点。
- 能整理实验结论：`a` 与 `F` 成正比，与 `m` 成反比。

## 核心知识点讲解

### 1. 实验目的

加速度描述运动状态变化快慢，力是改变运动状态的原因，质量影响物体运动状态改变的难易程度。

本实验探究：

```text
a 与 F、m 的定量关系
```

### 2. 控制变量法

因为加速度可能同时受力和质量影响，所以要控制变量：

- 保持质量 `m` 不变，研究 `a` 与 `F` 的关系。
- 保持合力 `F` 不变，研究 `a` 与 `m` 的关系。

### 3. 实验装置和基本思路

常用小车、长木板、滑轮、槽码、纸带、打点计时器。

槽码通过细线牵引小车运动。若槽码质量远小于小车质量，可近似认为：

```text
小车受到的拉力 ≈ 槽码重力
```

### 4. 平衡阻力

小车运动时有摩擦和纸带阻力。为了让小车所受合力近似等于细线拉力，需要把木板一端垫高。

调节到小车不受牵引时能拖着纸带匀速运动，说明阻力已被平衡。

### 5. 加速度测量

方法一：初速度为 0 的匀加速运动：

```text
x = 1/2 at^2
a = 2x/t^2
```

方法二：用打点计时器纸带求加速度。

方法三：若两辆小车运动时间相同：

```text
a1/a2 = x1/x2
```

### 6. 数据处理

研究 `a` 与 `F`：

- 以 `F` 为横轴、`a` 为纵轴。
- 若图像是过原点直线，说明 `a ∝ F`。

研究 `a` 与 `m`：

- 若猜想 `a ∝ 1/m`，应作 `a-1/m` 图像。
- 若图像是过原点直线，说明 `a ∝ 1/m`。

### 7. 实验结论

```text
质量一定时，加速度与合力成正比。
合力一定时，加速度与质量成反比。
```

即：

```text
a ∝ F/m
```

## 重点梳理

- 实验方法：控制变量法。
- `a-F` 过原点直线：`a ∝ F`。
- `a-1/m` 过原点直线：`a ∝ 1/m`。
- 平衡阻力是为了使小车所受合力近似等于拉力。
- 槽码质量应远小于小车质量。

## 难点突破

### 难点 1：为什么不直接作 `a-m` 图像判断

反比例图像是曲线，不如直线明显。作 `a-1/m` 图像更容易判断。

### 难点 2：为什么要平衡阻力

未平衡阻力时：

```text
F合 = F拉 - F阻
```

此时实验研究的不是 `a` 与拉力的真实关系。

### 难点 3：槽码重力为什么只是近似等于拉力

槽码也在加速运动，细线拉力小于槽码重力。槽码质量远小于小车质量时误差较小。

## 例题讲解

### 例题 1：由位移和时间求加速度

小车从静止开始运动，`2 s` 内位移 `1.6 m`。

```text
a = 2x/t^2 = 2×1.6/2^2 = 0.8 m/s^2
```

### 例题 2：判断 `a` 与 `F`

质量一定时，拉力变为原来的 2 倍，加速度也变为 2 倍，说明 `a` 与 `F` 成正比。

### 例题 3：判断 `a` 与 `m`

拉力一定时，若 `m` 变为原来的 2 倍，`a` 变为原来的 1/2，说明 `a` 与 `m` 成反比。

## 易错点整理

- 不控制变量，导致规律判断混乱。
- 未平衡阻力，图像不过原点。
- 把 `a-m` 曲线误认为没有规律。
- 忘记把质量单位统一。
- 纸带计数点间隔数错。

## 考点考证点整理

- 控制变量法。
- 平衡阻力的目的和判断。
- `a-F` 图像与 `a-1/m` 图像。
- 加速度测量方法。
- 实验误差来源。

## 练习题

### 基础题

1. 本实验采用什么方法？
2. 研究 `a` 与 `F` 时应保持什么不变？
3. 研究 `a` 与 `m` 时应保持什么不变？
4. `a-F` 图像过原点直线说明什么？
5. 为什么作 `a-1/m` 图像？

### 巩固题

6. 小车从静止开始，`3 s` 内位移 `4.5 m`，求加速度。
7. 质量不变时，合力变为原来的 3 倍，加速度如何变化？
8. 合力不变时，质量变为原来的 4 倍，加速度如何变化？

### 提升题

9. 若两辆小车运动时间相同，位移分别为 `0.40 m` 和 `0.60 m`，求加速度之比。
10. `a-F` 图像在横轴上有正截距，可能说明什么？

## 练习题答案

1. 控制变量法。
2. 质量。
3. 合力或拉力。
4. 质量一定时，加速度与合力成正比。
5. 反比例关系转化为正比例直线关系，便于判断。
6. `a = 2x/t^2 = 2×4.5/9 = 1 m/s^2`。
7. 变为原来的 3 倍。
8. 变为原来的 1/4。
9. `a1/a2 = x1/x2 = 0.40/0.60 = 2/3`。
10. 可能阻力未平衡或平衡不足。
