# 第三节 氧化还原反应

## 本节学习目标

学完本节后，应能从得氧失氧、化合价升降和电子转移三个层次认识氧化还原反应；能判断一个反应是否为氧化还原反应；能说明氧化反应、还原反应同时发生；能根据化合价变化判断氧化剂、还原剂、被氧化和被还原的物质；能用电子得失或共用电子对偏移解释氧化还原反应本质；能联系生产生活说明氧化还原反应的价值和风险。

源文核心知识点清单：

- 初中“得氧/失氧”只是氧化还原认识的起点，不能覆盖所有氧化还原反应。
- 氧化还原反应的重要特征是反应前后有元素化合价变化。
- 氧化反应和还原反应在同一反应中同时发生。
- 化合价升高对应氧化反应，化合价降低对应还原反应。
- 氧化还原反应的本质是电子转移，包括电子得失和共用电子对偏移。
- 氧化剂得到电子或电子对偏向，化合价降低，本身被还原；还原剂失去电子或电子对偏离，化合价升高，本身被氧化。
- 常见氧化剂和还原剂要能识别，并能分析汽车尾气催化转化、金属冶炼、钢铁锈蚀、食物腐败等情境。
- 四种基本反应类型与氧化还原反应有交叉关系：置换反应一定是，复分解反应一定不是，化合/分解反应可能是。

## 核心知识点讲解

### 一、知识对象与物质情境

本节研究“反应中电子怎样转移”。物质分类和离子反应帮助我们从类别、离子角度看反应；氧化还原反应进一步从化合价和电子角度揭示变化本质。燃烧、金属冶炼、钢铁锈蚀、食物腐败、光合作用、燃料电池、汽车尾气处理等，都与氧化还原反应有关。

### 二、核心概念与物质分类

1. 氧化还原反应

有元素化合价发生变化的反应叫氧化还原反应。初中把得氧叫氧化、失氧叫还原；高中要扩大为“化合价升降”和“电子转移”。

2. 氧化反应与还原反应

- 元素化合价升高，物质发生氧化反应，被氧化。
- 元素化合价降低，物质发生还原反应，被还原。
- 氧化和还原必然同时发生，因为一个物质失去电子或电子对偏离，必须有另一个物质得到电子或电子对偏向。

3. 氧化剂和还原剂

- 氧化剂：得到电子或电子对偏向的物质，所含元素化合价降低，本身被还原，具有氧化性。
- 还原剂：失去电子或电子对偏离的物质，所含元素化合价升高，本身被氧化，具有还原性。

常见氧化剂：$O_{2}$、$Cl_{2}$、浓硫酸、$HNO_{3}$、$KMnO_{4}$、$FeCl_{3}$等。

常见还原剂：活泼金属单质如$Al$、$Zn$、$Fe$，以及$C$、$H_{2}$、$CO$、$KI$等。

### 三、关键规律、反应原理与方程式

1. 从得氧失氧到化合价升降

例1：

$$
2CuO + C \xrightarrow{高温} 2Cu + CO_{2}\uparrow
$$

$C$得到氧生成$CO_{2}$，发生氧化反应，碳元素化合价由0升高到+4；$CuO$失去氧生成$Cu$，发生还原反应，铜元素由+2降低到0。

例2：

$$
Fe_{2}O_{3} + 3CO \xrightarrow{高温} 2Fe + 3CO_{2}
$$

$CO$发生氧化反应，$C$由+2升高到+4；$Fe_{2}O_{3}$发生还原反应，$Fe$由+3降低到0。

2. 没有得氧失氧，也可能是氧化还原反应

$$
Fe + CuSO_{4} = FeSO_{4} + Cu
$$

$Fe$由0价升高到+2价，被氧化；$Cu$由+2价降低到0价，被还原。虽然没有得氧失氧，但有化合价变化，所以是氧化还原反应。

3. 氧化还原反应本质：电子转移

$Na$与$Cl_{2}$反应：

$$
2Na + Cl_{2} = 2NaCl
$$

$Na$失去电子形成$Na^{+}$，化合价从0升高到+1，被氧化，是还原剂；$Cl_{2}$得到电子形成$Cl^{-}$，化合价从0降低到-1，被还原，是氧化剂。这类反应发生电子得失。

$H_{2}$与$Cl_{2}$反应：

$$
H_{2} + Cl_{2} \xrightarrow{点燃} 2HCl
$$

$H$和$Cl$形成共用电子对，但共用电子对偏向$Cl$、偏离$H$，氢元素化合价升高，氯元素化合价降低。这类反应发生共用电子对偏移。

4. 氧化剂、还原剂关系

例：

$$
CuO + H_{2} \xrightarrow{\Delta} Cu + H_{2}O
$$

$CuO$中$Cu$由+2价降为0价，$CuO$是氧化剂，被还原；$H_{2}$中$H$由0价升为+1价，$H_{2}$是还原剂，被氧化。

5. 四种基本反应类型与氧化还原反应

- 置换反应一定是氧化还原反应，因为单质与化合物互换时必有元素化合价变化。
- 复分解反应一定不是氧化还原反应，因为只是离子互换，元素化合价不变。
- 化合反应可能是，如$2Na + Cl_{2} = 2NaCl$是；$CaO + H_{2}O = Ca(OH)_{2}$不是。
- 分解反应可能是，如$2KClO_{3}$分解制$O_{2}$是；$CaCO_{3}$分解不是。

6. 汽车尾气催化转化

$CO$和$NO$在催化剂作用下生成$CO_{2}$和$N_{2}$：

$$
2CO + 2NO \xrightarrow{催化剂} 2CO_{2} + N_{2}
$$

$C$由+2升高到+4，$CO$是还原剂；$N$由+2降低到0，$NO$是氧化剂。该反应能把有毒$CO$和$NO$转化为相对无害的$CO_{2}$和$N_{2}$，减少尾气污染。

### 四、典型转化模型与分析方法

1. 氧化还原判断模型

标化合价 -> 找变化 -> 判类型。只要有元素化合价变化，就是氧化还原反应；没有变化就不是。

2. 氧化剂/还原剂判断模型

升失氧还，降得还氧。

含义展开：

- 化合价升高，失去电子或电子对偏离，发生氧化反应，是还原剂。
- 化合价降低，得到电子或电子对偏向，发生还原反应，是氧化剂。

3. 需要加入氧化剂还是还原剂

目标元素化合价要升高，说明该物质要被氧化，需要加入氧化剂。目标元素化合价要降低，说明该物质要被还原，需要加入还原剂。

例：$Fe^{2+}$ -> $Fe^{3+}$化合价升高，需要氧化剂；$CuO$ -> $Cu$化合价降低，需要还原剂。

4. 多元素变化分析模型

复杂方程式中先找所有元素化合价，再抓真正变化的元素。没有变化的元素不参与氧化还原判断。若同一物质中某元素既升又降，可能发生歧化或自身氧化还原反应。

### 五、实验现象、装置与证据

1. 教材思考讨论中的证据

$CuO$与$C$、$Fe_{2}O_{3}$与$CO$两个反应说明：得氧和失氧同时出现，氧化反应和还原反应不能孤立发生；化合价升降能统一解释得氧失氧。

2. $NaCl$形成示意图

Na原子最外层1个电子，易失去形成$Na^{+}$；Cl原子最外层7个电子，易得到电子形成$Cl^{-}$。该图承载的核心知识点是“电子得失导致化合价变化”。

3. $HCl$形成过程

H和Cl没有完全得失电子，而是形成共用电子对；由于Cl吸引电子能力较强，电子对偏向Cl，导致H显+1价、Cl显-1价。该过程说明电子转移不只包括得失，也包括偏移。

4. 科学史话

氧化还原概念经历了“得氧失氧 -> 化合价升降 -> 电子转移”的发展过程。启示是：化学概念会随着证据和理论的发展而完善，学习时要注意概念的适用范围。

5. 生活生产证据

金属冶炼、电镀、燃料燃烧、光合作用、自燃、食物腐败、钢铁锈蚀都是氧化还原反应。它们有正面价值，也可能带来损失；掌握规律后可以趋利避害，如防锈、尾气净化、污水处理。

### 六、题型应用与迁移

- 判断氧化还原反应：标化合价，看是否变化。
- 判断氧化剂和还原剂：化合价降的是氧化剂，升的是还原剂。
- 判断$HCl$、$H_{2}O_{2}$等在不同反应中的作用：同一物质在不同反应中可能作氧化剂、还原剂，甚至既作氧化剂又作还原剂。
- 根据目标价态变化判断需加入氧化剂还是还原剂。
- 分析实际情境，如铝热反应、黑火药、火法炼锌、重铬酸根废水处理、高铁酸钠制备。

## 重点梳理

- 氧化还原反应的特征：元素化合价发生变化。
- 氧化还原反应的本质：电子转移，包括电子得失和共用电子对偏移。
- 化合价升高 -> 被氧化 -> 发生氧化反应 -> 作还原剂。
- 化合价降低 -> 被还原 -> 发生还原反应 -> 作氧化剂。
- 氧化剂具有氧化性，本身被还原；还原剂具有还原性，本身被氧化。
- 置换反应一定是氧化还原反应；复分解反应一定不是氧化还原反应；化合、分解反应可能是。
- 判断题必须以化合价变化为依据，不能只靠是否有氧元素参加。

## 难点突破

1. “氧化剂被还原”为什么听起来相反？

氧化剂的作用是氧化别人，即让别人失电子、化合价升高；它自己得到电子，所以自己被还原。还原剂同理：还原别人，自己被氧化。

2. 没有氧元素的反应怎么判断？

看化合价。例如$Fe$ + $CuSO_{4}$ = $FeSO_{4}$ + $Cu$没有得氧失氧，但$Fe$升价、$Cu$降价，是氧化还原反应。

3. 如何判断$HCl$作氧化剂还是还原剂？

看$HCl$中哪种元素变价。若$H^{+}$变为$H_{2}$，氢元素从+1降到0，$HCl$作氧化剂；若$Cl^{-}$变为$Cl_{2}$，氯元素从-1升到0，$HCl$作还原剂；若化合价都不变，如中和反应，则$HCl$既不是氧化剂也不是还原剂。

4. $H_{2}O_{2}$为什么既能作氧化剂又能作还原剂？

$H_{2}O_{2}$中氧元素为-1价，处于中间价态。它可降到-2价，如生成$H_{2}O$时作氧化剂；也可升到0价，如生成$O_{2}$时作还原剂；分解时一部分氧降价、一部分氧升价，既作氧化剂又作还原剂。

## 例题讲解

### 例题

分析反应：

$$
3ClO^{-} + 2Fe^{3+} + 10OH^{-} = 2FeO_{4}^{2-} + 3Cl^{-} + 5H_{2}O
$$

指出元素化合价变化、氧化剂和还原剂。

### 分析

先标变化元素。$ClO^{-}$中O为-2价，整体为-1价，所以Cl为+1价；$Cl^{-}$中Cl为-1价，氯元素化合价降低。$Fe^{3+}$中$Fe$为+3价；$FeO_{4}^{2-}$中O为-2价，4个O共-8价，整体为-2价，所以$Fe$为+6价，铁元素化合价升高。

### 答案

Cl：+1 -> -1，化合价降低，$ClO^{-}$得到电子，是氧化剂，本身被还原。

$Fe$：+3 -> +6，化合价升高，$Fe^{3+}$失去电子，是还原剂，本身被氧化。

### 反思

复杂离子方程式不要被$OH^{-}$、$H_{2}O$等干扰。先用化合价规则找到变价元素，再根据升降判断氧化剂和还原剂。

## 易错点整理

- 认为只有有氧元素参加的反应才是氧化还原反应。应以化合价变化为准。
- 把“被氧化”和“氧化剂”混淆。被氧化的是还原剂，氧化剂自己被还原。
- 判断氧化剂、还原剂时看产物而不看反应物。氧化剂和还原剂都是反应物。
- 忘记单质中元素化合价为0。
- 对多原子离子化合价计算错误，如$FeO_{4}^{2-}$中$Fe$为+6。
- 认为化合反应或分解反应一定是氧化还原反应。它们只是可能是。
- 忽略同一物质在不同反应中作用不同，如$HCl$、$H_{2}O_{2}$。

## 考点考证点整理

### 考点一：氧化还原反应判断

- 出题思路：给出若干反应，要求判断是否属于氧化还原反应。
- 关键条件：反应前后是否有元素化合价变化。
- 解答要点：先标变价元素，再写“有化合价变化，所以是氧化还原反应”或“无化合价变化，所以不是”。
- 易扣分点：只看是否有氧元素；把所有化合反应、分解反应都判断为氧化还原反应。

### 考点二：氧化剂、还原剂判断

- 出题思路：给出方程式，要求指出氧化剂、还原剂、被氧化和被还原的物质。
- 关键条件：氧化剂是反应物中化合价降低的物质；还原剂是反应物中化合价升高的物质。
- 解答要点：写清变价元素、价态变化、物质作用，如“$Fe$由0升到+2，作还原剂，被氧化”。
- 易扣分点：把生成物写成氧化剂/还原剂；把“氧化剂被还原、还原剂被氧化”说反。

### 考点三：需要加入氧化剂还是还原剂

- 出题思路：给出离子或物质转化，如$Fe^{2+} \rightarrow Fe^{3+}$，判断需要氧化剂还是还原剂。
- 关键条件：目标元素价态升高说明被氧化，需要氧化剂；价态降低说明被还原，需要还原剂。
- 解答要点：先标目标元素价态变化，再写需要加入的试剂类型。
- 易扣分点：不看价态，只凭“加氧/去氧”判断。

### 考点四：同一物质在不同反应中的作用

- 出题思路：考查$HCl$、$H_{2}O_{2}$、$H_{2}O$等在不同反应中作氧化剂、还原剂或二者兼有。
- 关键条件：同一物质中参与变价的元素不同，作用就可能不同。
- 解答要点：逐个反应标价，说明哪种元素升降；$H_{2}O_{2}$中$O$为-1价，既可升到0价，也可降到-2价。
- 易扣分点：机械记忆某物质固定作氧化剂或还原剂，不结合具体反应。

### 考点五：材料情境中的氧化还原分析

- 出题思路：以铝热反应、黑火药、废水处理、高铁酸钠、火法炼锌、汽车尾气处理等材料设问。
- 关键条件：材料信息通常给出方程式、用途或部分产物，必须抓变价元素。
- 解答要点：补全方程式后标价，指出氧化剂/还原剂，并联系材料说明用途或环保意义。
- 易扣分点：读材料只提取用途，不做价态分析；复杂离子中中心元素价态计算错误。

## 练习题答案

1. 方程式：

$$
2H_{2}O_{2} \xrightarrow{MnO_{2}} 2H_{2}O + O_{2}\uparrow
$$

$$
2KClO_{3} \xrightarrow{MnO_{2},\Delta} 2KCl + 3O_{2}\uparrow
$$

$$
2KMnO_{4} \xrightarrow{\Delta} K_{2}MnO_{4} + MnO_{2} + O_{2}\uparrow
$$

共同点：都属于分解反应，也都是氧化还原反应，因为有元素化合价变化。

2. $C$由0价升高到+2价，被氧化，是还原剂；$H_{2}O$中H由+1价降低到0价，$H_{2}O$是氧化剂。

3. 铝热反应：

$$
2Al + Fe_{2}O_{3} \xrightarrow{高温} Al_{2}O_{3} + 2Fe
$$

$Al$由0价升高到+3价，是还原剂；$Fe_{2}O_{3}$中$Fe$由+3价降低到0价，$Fe_{2}O_{3}$是氧化剂，被还原。

4. ①金属冶炼、②钢铁锈蚀、③食物腐败涉及氧化还原反应；④钟乳石形成主要涉及碳酸盐与碳酸氢盐转化，通常不按氧化还原反应处理。判断依据是是否有元素化合价变化。

5. 一定是氧化还原反应的是③置换反应；一定不是氧化还原反应的是④复分解反应；可能是氧化还原反应的是①化合反应和②分解反应。关键是不要按反应类型机械判断，要回到化合价变化。

6. 属于氧化还原反应的是C：$WO_{3} + 3H_{2} \xrightarrow{高温} W + 3H_{2}O$，其中$W$由+6降为0，$H$由0升为+1。A、B、D中元素化合价不变，不属于氧化还原反应。

7. $HCl$作还原剂的是C，因为$Cl$由-1升到0生成$Cl_{2}$；$HCl$作氧化剂的是B，因为$H$由+1降到0生成$H_{2}$。A、D中各元素化合价不变，$HCl$既不是氧化剂也不是还原剂。

8. 需要加入还原剂的是C：$CuO \rightarrow Cu$，$Cu$由+2降到0，需要被还原。A无价态变化；B中$Cu$升价，需要氧化剂；D中$C$由+2升到+4，需要氧化剂。

9. 正确说法是D：$H_{2}O$中的氢元素被还原。$NaH$中$H$为-1价，生成$H_{2}$中$H$为0价，$NaH$中氢被氧化，$NaH$作还原剂；$H_{2}O$中$H$由+1降到0，被还原，$H_{2}O$作氧化剂。该反应不属于置换反应，也不属于复分解反应。

10. （1）$H_{2}$中$H$由0升到+1，$O_{2}$中$O$由0降到-2，$O_{2}$是氧化剂，$H_{2}$是还原剂。（2）$P$由0升到+5，$O$由0降到-2，$O_{2}$是氧化剂，$P$是还原剂。（3）$Fe$由0升到+2，$H^{+}$由+1降到0，$H_{2}SO_{4}$表现氧化性，$Fe$是还原剂。（4）$Hg$由+2降到0，$O$由-2升到0，$HgO$既作氧化剂又作还原剂。

11. $H_{2}O_{2} + H_{2}S = 2H_{2}O + S\downarrow$中，$H_{2}O_{2}$中$O$由-1降到-2，作氧化剂；$H_{2}O_{2} + Cl_{2} = 2HCl + O_{2}$中，$H_{2}O_{2}$中$O$由-1升到0，作还原剂；$2H_{2}O_{2} \xrightarrow{MnO_{2}} 2H_{2}O + O_{2}\uparrow$中，$H_{2}O_{2}$既作氧化剂又作还原剂。

12. $ClO^{-}$中$Cl$为+1价，生成$Cl^{-}$后为-1价，化合价降低，$ClO^{-}$是氧化剂；$Fe^{3+}$中$Fe$为+3价，$FeO_{4}^{2-}$中$Fe$为+6价，化合价升高，$Fe^{3+}$是还原剂。解题关键是会计算复杂离子中中心元素价态。
