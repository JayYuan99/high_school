# 第一节 原子结构与元素周期表

## 本节学习目标

- 理解原子由原子核和核外电子构成，掌握质子数、中子数、质量数之间的关系。
- 知道核素、同位素的含义。
- 理解元素周期表的周期、族、原子序数等基本结构。
- 能从原子结构角度认识元素在周期表中的位置和性质。

## 核心知识点讲解

### 一、知识对象与物质情境

元素周期表不是简单的元素名单，而是把元素按原子结构和性质规律排列形成的系统工具。学习本节，就是要把“原子结构”和“元素位置”联系起来。

### 二、核心概念与物质分类

| 概念 | 含义 | 关系 |
| --- | --- | --- |
| 原子序数 | 元素在周期表中的序号 | 原子序数 = 质子数 |
| 质量数 | 质子数与中子数之和 | A = Z + N |
| 核素 | 具有一定质子数和中子数的一种原子 | 如 ^1_1H、^2_1H |
| 同位素 | 质子数相同、中子数不同的同种元素原子 | 氕、氘、氚 |

### 三、关键规律、反应原理与方程式

质量数关系：

$$
A = Z + N
$$

其中 A 为质量数，Z 为质子数，N 为中子数。

元素周期表中，元素原子的核外电子层数通常与周期序数有关，最外层电子数与主族序数有关。

### 四、典型转化模型与分析方法

判断元素位置的基本思路：

```text
原子序数 -> 核外电子排布 -> 电子层数和最外层电子数 -> 周期和族
```

例如 Na 原子核外电子排布可表示为 2、8、1，所以位于第三周期、第 IA 族。

### 五、实验现象、装置与证据

教材通过碱金属等元素性质比较，引导认识同主族元素性质相似但有递变。实验现象是结构规律的证据，而不是孤立事实。

### 六、题型应用与迁移

常考质量数计算、同位素判断、元素位置判断、原子结构与元素性质关系、周期表中元素信息读取。

## 重点梳理

- 原子序数 = 质子数 = 核电荷数。
- 质量数 = 质子数 + 中子数。
- 同位素属于同一种元素，化学性质相似。
- 周期序数通常等于电子层数。
- 主族元素最外层电子数通常等于主族序数。

## 难点突破

- 同位素不是不同元素，而是同种元素的不同核素。
- 相对原子质量不是质量数，质量数是整数，相对原子质量通常是平均值。
- 周期表位置判断不能只背元素名称，要会从原子结构推出。

## 例题讲解

### 例题

某原子质子数为 11，中子数为 12。求其质量数，并判断它是什么元素。

### 分析

质量数：

$$
A = Z + N = 11 + 12 = 23
$$

质子数为 11，即原子序数为 11，对应 Na。

### 答案

质量数为 23，是钠元素的一种核素，可表示为 ^{23}_{11}Na。

### 反思

元素种类由质子数决定，不由中子数决定。

## 易错点整理

| 易错点 | 错因 | 防错提醒 |
| --- | --- | --- |
| 把中子数当原子序数 | 概念混淆 | 原子序数看质子数 |
| 同位素当不同元素 | 忽视质子数相同 | 同位素属于同种元素 |
| 质量数写成相对原子质量 | 概念混淆 | 质量数是质子数加中子数 |

## 考点考证点整理

- A、Z、N 的计算。
- 核素和同位素判断。
- 元素周期表结构。
- 原子结构与周期表位置关系。

## 练习题

1. 某原子质子数为 17，中子数为 18，质量数是多少？
2. ^1_1H、^2_1H、^3_1H 互为什么关系？
3. 原子序数为 12 的元素位于第几周期？最外层电子数是多少？
4. 元素种类由什么决定？

## 练习题答案

1. 35。
2. 互为同位素。
3. Mg 的电子排布为 2、8、2，位于第三周期，最外层电子数为 2。
4. 由质子数决定。
