# 第八单元 古诗词诵读 涉江采芙蓉

                                 涉江采芙蓉

                                 《古诗十九首》

                                        b
                             涉江采芙蓉，兰泽 多芳草。
                                   c
                             采之欲遗 谁？所思在远道。
                             还顾望旧乡，长路漫浩浩。
                                        d
                             同心而离居，忧伤以 终老。

               东汉中后期，游宦、游学之风盛行，大批士子远离故乡。作为文人五言诗滥
            觞的《古诗十九首》反映的就是这一现实，诗中充满了羁旅怀乡的思绪。这首
            诗的主人公采撷美草香花要送给“同心”的人，但他思念的人还在“旧乡”，长
            路漫漫，欲赠不能，只能让忧伤陪伴自己终老。全诗字里行间流露着漂泊他乡
            和分隔两地的无奈。诵读这首诗，体会其“浅而能深、近而能远”（胡应麟《诗
            薮·内编》卷一）的特色。

         a 选自《文选》卷二十九（上海古籍出版社1986年版）。    c〔遗（wèi）〕赠送。
           芙蓉，荷花。                       d〔以〕连词，表示结果。
         b〔兰泽〕长着兰草的低湿之地。
