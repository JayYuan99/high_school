# 第一节 铁及其化合物

## 本节学习目标

- 掌握铁单质、Fe^{2+}、Fe^{3+} 及常见铁的氧化物、氢氧化物的性质。
- 能从价态变化角度理解铁及其化合物之间的转化。
- 会用 KSCN 溶液检验 Fe^{3+}。
- 能书写铁及其重要化合物的典型反应方程式。

## 核心知识点讲解

### 一、知识对象与物质情境

铁是使用最广泛的金属材料之一。自然界中铁主要以 +2 价和 +3 价化合物存在，铁的学习重点是“金属单质性质 + 变价元素转化”。

### 二、核心概念与物质分类

| 物质或离子 | 类别 | 典型性质 |
| --- | --- | --- |
| Fe | 金属单质 | 还原性，可与酸、盐溶液、非金属反应 |
| FeO | 碱性氧化物 | 铁为 +2 价 |
| Fe_2O_3 | 碱性氧化物 | 铁为 +3 价，红棕色 |
| Fe(OH)_2 | 碱 | 白色，易被氧化 |
| Fe(OH)_3 | 碱 | 红褐色沉淀 |
| Fe^{2+} | 亚铁离子 | 有还原性，易被氧化 |
| Fe^{3+} | 铁离子 | 有氧化性，遇 KSCN 显红色 |

### 三、关键规律、反应原理与方程式

铁与盐酸反应：

$$
Fe + 2HCl = FeCl_2 + H_2\uparrow
$$

铁与 CuSO_4 溶液反应：

$$
Fe + CuSO_4 = FeSO_4 + Cu
$$

Fe^{2+} 被氧化为 Fe^{3+}：

$$
2Fe^{2+} + Cl_2 = 2Fe^{3+} + 2Cl^-
$$

Fe^{3+} 与 KSCN 溶液显红色，可用于检验 Fe^{3+}。

### 四、典型转化模型与分析方法

铁元素常见价态转化：

```text
Fe(0) -> Fe2+ -> Fe3+
```

分析时先标铁元素价态，再判断加入的物质是氧化剂还是还原剂。

### 五、实验现象、装置与证据

向 FeCl_3 溶液中滴加 KSCN 溶液，溶液变红，说明存在 Fe^{3+}。向 FeSO_4 溶液中滴加 NaOH 溶液，先生成白色 Fe(OH)_2 沉淀，随后迅速变灰绿，最终变红褐色 Fe(OH)_3。

### 六、题型应用与迁移

常考铁离子检验、Fe^{2+}/Fe^{3+} 转化、铁与酸或盐溶液反应、铁的化合物颜色判断和实验现象解释。

## 重点梳理

- 铁单质常表现还原性。
- Fe^{2+} 易被氧化为 Fe^{3+}。
- Fe^{3+} 遇 KSCN 溶液显红色。
- Fe(OH)_2 易被空气中 O_2 氧化为 Fe(OH)_3。
- 铁及其化合物题目要抓价态变化。

## 难点突破

- Fe 与弱氧化性酸反应通常生成 Fe^{2+}，不是 Fe^{3+}。
- Fe^{3+} 检验现象是溶液变红，不是生成红色沉淀。
- Fe(OH)_2 的白色沉淀很容易被氧化，实验中常看到颜色连续变化。

## 例题讲解

### 例题

某溶液中可能含 Fe^{2+} 或 Fe^{3+}。加入 KSCN 溶液后变红，说明什么？

### 分析

KSCN 是 Fe^{3+} 的常用检验试剂，Fe^{3+} 与 SCN^- 形成红色络合物。

### 答案

说明原溶液中含有 Fe^{3+}。

### 反思

若要检验 Fe^{2+}，通常需要先排除 Fe^{3+}，再加入氧化剂观察是否产生 Fe^{3+} 的特征反应。

## 易错点整理

| 易错点 | 错因 | 防错提醒 |
| --- | --- | --- |
| Fe 与盐酸产物写 FeCl_3 | 忽视氧化性强弱 | 盐酸中 H^+ 通常只把 Fe 氧化成 Fe^{2+} |
| KSCN 检验说成沉淀 | 现象记忆不准 | 现象是溶液显红色 |
| Fe(OH)_2 颜色只写白色 | 忽视空气氧化 | 要说明白色迅速变灰绿再变红褐 |

## 考点考证点整理

- 铁及铁离子性质。
- Fe^{2+} 与 Fe^{3+} 转化。
- Fe^{3+} 检验。
- 氧化还原视角分析铁元素价态。

## 练习题

1. 写出 Fe 与盐酸反应的化学方程式。
2. 如何检验 Fe^{3+}？
3. FeSO_4 溶液中加入 NaOH 后沉淀颜色为什么会变化？
4. Fe^{2+} 转化为 Fe^{3+} 时，铁元素发生氧化还是还原？

## 练习题答案

1. $$Fe + 2HCl = FeCl_2 + H_2\uparrow$$
2. 加入 KSCN 溶液，若溶液变红，说明含 Fe^{3+}。
3. 先生成 Fe(OH)_2 白色沉淀，随后被空气中 O_2 氧化为红褐色 Fe(OH)_3。
4. 氧化。Fe 元素化合价从 +2 升高到 +3。
