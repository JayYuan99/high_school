# 第二节 金属材料

## 本节学习目标

- 理解合金的概念，知道铁合金、铝合金和新型合金的特点。
- 能从组成、结构、性能、用途的关系理解金属材料。
- 掌握物质的量在化学方程式计算中的基本应用。
- 会根据反应方程式中的计量关系进行简单计算。

## 核心知识点讲解

### 一、知识对象与物质情境

日常生产生活中使用的金属材料多数不是纯金属，而是合金。例如钢铁、铝合金、不锈钢等。合金通过改变组成改善强度、硬度、耐腐蚀性等性能。

### 二、核心概念与物质分类

| 材料 | 组成特点 | 主要性能 |
| --- | --- | --- |
| 生铁 | 含碳量较高的铁合金 | 硬而脆 |
| 钢 | 含碳量较低的铁合金 | 韧性和机械性能较好 |
| 不锈钢 | 含 Cr、Ni 等元素 | 耐腐蚀 |
| 铝合金 | 以 Al 为主，加入其他元素 | 密度小、强度较高 |
| 新型合金 | 具有特殊功能 | 耐高温、记忆、储氢等 |

### 三、关键规律、反应原理与方程式

金属材料学习强调：

```text
组成 -> 结构 -> 性能 -> 用途
```

化学方程式计算中，方程式系数表示物质的量之比。例如：

$$
Fe + H_2SO_4 = FeSO_4 + H_2\uparrow
$$

系数比说明 1 mol Fe 与 1 mol H_2SO_4 反应生成 1 mol H_2。

### 四、典型转化模型与分析方法

化学方程式计算步骤：

```text
写方程式 -> 找已知量并转化为 n -> 按系数比求未知 n -> 转换成题目要求的量
```

### 五、实验现象、装置与证据

铝表面有致密氧化膜，常温下具有一定耐腐蚀性。铝既能与酸反应，也能与强碱溶液反应，体现出特殊性质。

### 六、题型应用与迁移

常考合金性质判断、金属材料用途选择、铝的性质、化学方程式计算、气体体积与物质的量结合计算。

## 重点梳理

- 合金是由金属与金属或非金属熔合形成的具有金属特性的物质。
- 合金性能通常优于纯金属，应用更广。
- 钢和生铁都是铁碳合金，主要差别是含碳量不同。
- 铝合金密度小、强度高，常用于航空航天和交通工具。
- 方程式计算本质是物质的量比例计算。

## 难点突破

- 合金通常是混合物，不是化合物。
- “性能决定用途”要结合具体需求，如强度、密度、耐腐蚀性、导电性。
- 方程式计算中不能直接用质量比代替物质的量比，必须先转化为 n 或利用摩尔质量正确换算。

## 例题讲解

### 例题

250 mL 2.0 mol/L H_2SO_4 与足量铁完全反应，可生成 H_2 多少 mol？

### 分析

先求 H_2SO_4 的物质的量：

$$
n(H_2SO_4)=2.0mol/L \times 0.250L=0.50mol
$$

反应：

$$
Fe + H_2SO_4 = FeSO_4 + H_2\uparrow
$$

H_2SO_4 与 H_2 系数比为 1:1。

### 答案

$$
n(H_2)=0.50mol
$$

### 反思

“足量铁”说明 H_2SO_4 完全反应，计算以 H_2SO_4 为准。

## 易错点整理

| 易错点 | 错因 | 防错提醒 |
| --- | --- | --- |
| 把合金当纯净物 | 忽视组成 | 合金通常是混合物 |
| 方程式计算直接比质量 | 不懂系数含义 | 系数比是物质的量比 |
| 忘记 mL 转 L | 单位错误 | 浓度计算体积用 L |

## 考点考证点整理

- 合金概念和金属材料性能。
- 铁合金、铝合金、新型合金用途。
- 物质的量在方程式计算中的应用。
- 限量反应物判断。

## 练习题

1. 生铁和钢的主要组成差异是什么？
2. 为什么铝合金适合制造飞机部件？
3. 0.20 mol Fe 与足量盐酸反应，最多生成 H_2 多少 mol？
4. 合金一定是化合物吗？说明理由。

## 练习题答案

1. 主要差异是含碳量不同，生铁含碳量较高，钢含碳量较低。
2. 铝合金密度小、强度较高，且具有一定耐腐蚀性。
3. 由 $$Fe + 2HCl = FeCl_2 + H_2\uparrow$$ 可知 Fe 与 H_2 为 1:1，所以生成 0.20 mol H_2。
4. 不一定。合金通常是混合物，由多种成分熔合形成。
