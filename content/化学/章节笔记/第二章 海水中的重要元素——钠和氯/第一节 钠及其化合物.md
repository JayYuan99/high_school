# 第一节 钠及其化合物

## 本节学习目标

- 掌握钠单质的物理性质、保存方法和主要化学性质。
- 理解 Na_2O、Na_2O_2、Na_2CO_3、NaHCO_3 等钠的重要化合物的性质差异。
- 会书写钠及其化合物的典型反应方程式。
- 理解焰色试验的基本原理和操作注意事项。

## 核心知识点讲解

### 一、知识对象与物质情境

海水中含有大量钠元素，但自然界中的钠主要以化合态存在。金属钠非常活泼，钠的化合物广泛用于生活和工业，如食盐、纯碱、小苏打等。

### 二、核心概念与物质分类

| 物质 | 类别 | 关键性质 |
| --- | --- | --- |
| Na | 活泼金属单质 | 易失电子，强还原性 |
| Na_2O | 碱性氧化物 | 与水生成 NaOH |
| Na_2O_2 | 过氧化物 | 与水、CO_2 反应放出 O_2 |
| Na_2CO_3 | 盐，俗称纯碱 | 溶液显碱性，较稳定 |
| NaHCO_3 | 盐，俗称小苏打 | 受热易分解，可与酸反应 |

### 三、关键规律、反应原理与方程式

钠与水反应：

$$
2Na + 2H_2O = 2NaOH + H_2\uparrow
$$

钠在氧气中燃烧：

$$
2Na + O_2 \xrightarrow{\Delta} Na_2O_2
$$

过氧化钠与水、二氧化碳反应：

$$
2Na_2O_2 + 2H_2O = 4NaOH + O_2\uparrow
$$

$$
2Na_2O_2 + 2CO_2 = 2Na_2CO_3 + O_2
$$

碳酸氢钠受热分解：

$$
2NaHCO_3 \xrightarrow{\Delta} Na_2CO_3 + H_2O + CO_2\uparrow
$$

### 四、典型转化模型与分析方法

钠及其化合物可以按“单质 -> 氧化物 -> 碱 -> 盐”建立转化链：

```text
Na -> Na2O/Na2O2 -> NaOH -> Na2CO3/NaHCO3
```

分析时抓住钠元素通常表现 +1 价，金属钠单质易失电子，具有强还原性。

### 五、实验现象、装置与证据

钠与水反应的典型现象可概括为“浮、熔、游、响、红”：钠浮在水面，熔成小球，四处游动，发出响声，滴加酚酞后溶液变红。

焰色试验中，钠元素火焰呈黄色，钾元素火焰透过蓝色钴玻璃观察呈紫色。

### 六、题型应用与迁移

常考钠与水反应现象解释、Na_2CO_3 和 NaHCO_3 性质比较、Na_2O_2 与 CO_2/H_2O 反应计算、焰色试验判断。

## 重点梳理

- 钠保存在煤油中，原因是钠易与空气中 O_2、水蒸气反应。
- 钠与水反应生成 NaOH 和 H_2。
- Na_2O_2 既能与 H_2O 反应，也能与 CO_2 反应放出 O_2。
- NaHCO_3 受热易分解，Na_2CO_3 热稳定性较强。
- 焰色试验是元素性质，属于物理变化。

## 难点突破

- Na_2CO_3 俗称纯碱，但它不是碱，而是盐。
- Na_2O_2 中氧元素为 -1 价，不同于普通氧化物中的 -2 价。
- 焰色试验不是化学反应，不写化学方程式。

## 例题讲解

### 例题

如何鉴别 Na_2CO_3 固体和 NaHCO_3 固体？

### 分析

二者都能与酸反应放出 CO_2，但热稳定性不同。NaHCO_3 受热分解产生 CO_2，Na_2CO_3 较稳定。

### 答案

分别取少量固体加热，将产生的气体通入澄清石灰水。若石灰水变浑浊，说明产生 CO_2，该固体为 NaHCO_3；无明显变化的是 Na_2CO_3。

### 反思

鉴别题要找性质差异，不要选择二者都能发生的反应。

## 易错点整理

| 易错点 | 错因 | 防错提醒 |
| --- | --- | --- |
| 把纯碱当碱 | 被俗名误导 | Na_2CO_3 属于盐 |
| 忘记 Na_2O_2 放氧 | 当成普通氧化物 | 记住与水、CO_2 均产生 O_2 |
| 焰色试验前不洗铂丝 | 杂质干扰 | 用盐酸洗至无色火焰 |

- Na_2O_2 的反应和氧化还原分析。
- Na_2CO_3 与 NaHCO_3 性质比较。
- 焰色试验操作与颜色判断。

## 练习题

1. 写出钠与水反应的化学方程式。
2. Na_2O_2 与 CO_2 反应中，氧气来自哪种物质？
3. 为什么金属钠要保存在煤油中？
4. 设计一个实验区分 Na_2CO_3 和 NaHCO_3。

## 练习题答案

1. $$2Na + 2H_2O = 2NaOH + H_2\uparrow$$
2. 氧气来自 Na_2O_2 中 -1 价氧元素的氧化还原变化。
3. 防止钠与空气中的氧气和水蒸气反应。
4. 加热固体并将气体通入澄清石灰水，能使石灰水变浑浊的是 NaHCO_3。
